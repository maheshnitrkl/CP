📖 Theory

📊 Sorting Algorithm Comparison

Algorithm Best Average Worst Space Stable?
Bubble Sort O(n) O(n²) O(n²) O(1) ✅ Yes
Insertion Sort O(n) O(n²) O(n²) O(1) ✅ Yes
Quick Sort O(n log n) O(n log n) O(n²) O(log n) ❌ No
Merge Sort O(n log n) O(n log n) O(n log n) O(n) ✅ Yes
Counting Sort O(n+k) O(n+k) O(n+k) O(k) ✅ Yes

k = range of input values for Counting Sort

💡 When to Use What?
Bubble/Insertion: Small datasets, nearly sorted data
Quick Sort: General purpose, in-place (preferred in practice)
Merge Sort: When stability is required, or for linked lists
Counting Sort: When range of values is small (e.g., characters, small integers)

🫧 Bubble Sort

Repeatedly swap adjacent elements if they are in the wrong order. After each pass, the largest unsorted element "bubbles up" to its correct position.

Key Idea: Compare adjacent pairs, swap if needed. Repeat until no swaps are needed.

#include <iostream>
#include <vector>
using namespace std;

int main() {
    int n;
    cout << "Enter number of elements: ";
    cin >> n;
    
    vector<int> arr(n);
    cout << "Enter " << n << " elements: ";
    for (int i = 0; i < n; i++) {
        cin >> arr[i];
    }
    
    // We do n-1 passes. In each pass, the largest remaining element "bubbles" to the end.
    for (int i = 0; i < n - 1; i++) {
        bool swapped = false;
        // Compare adjacent elements. We stop early (n - i - 1) because the last 'i' elements are already sorted!
        for (int j = 0; j < n - i - 1; j++) {
            // If the left element is bigger than the right element, they are out of order
            if (arr[j] > arr[j + 1]) {
                swap(arr[j], arr[j + 1]); // Swap them!
                swapped = true;           // Remember that we had to make a swap
            }
        }
        // If we went through the whole array without making a single swap, it's completely sorted!
        if (!swapped) break;
    }
    
    cout << "Sorted array: ";
    for (int i = 0; i < n; i++) {
        cout << arr[i] << " ";
    }
    cout << endl;
    
    return 0;
}
if __name__ == "__main__":
    n = int(input("Enter number of elements: "))
    arr = list(map(int, input("Enter elements: ").split()))
    
    for i in range(len(arr) - 1):
        swapped = False
        for j in range(len(arr) - i - 1):
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
                swapped = True
        if not swapped:
            break
            
    print("Sorted array:", *arr)

📥 Insertion Sort

Build the sorted array one element at a time. Take each element and insert it into its correct position among the already-sorted elements.

Analogy: Like sorting playing cards in your hand — pick one card at a time and insert it in the right position.

#include <iostream>
#include <vector>
using namespace std;

int main() {
    int n;
    cout << "Enter number of elements: ";
    cin >> n;
    
    vector<int> arr(n);
    cout << "Enter " << n << " elements: ";
    for (int i = 0; i < n; i++) {
        cin >> arr[i];
    }
    
    // Assume the first element (index 0) is already "sorted". Start from index 1.
    for (int i = 1; i < n; i++) {
        int key = arr[i]; // The card we want to insert into the sorted part
        int j = i - 1;    // Start comparing with the element right before it
        
        // Move elements of the sorted part that are greater than the key
        // one position ahead to make room for the key.
        while (j >= 0 && arr[j] > key) {
            arr[j + 1] = arr[j]; // Shift element to the right
            j--;                 // Move backwards
        }
        // Place the key into its correct sorted position
        arr[j + 1] = key;
    }
    
    cout << "Sorted array: ";
    for (int i = 0; i < n; i++) {
        cout << arr[i] << " ";
    }
    cout << endl;
    
    return 0;
}
if __name__ == "__main__":
    n = int(input("Enter number of elements: "))
    print(f"Enter {n} elements separated by space: ")
    arr = list(map(int, input().split()))
    
    # Assume the first element (index 0) is already "sorted". Start from index 1.
    for i in range(1, len(arr)):
        key = arr[i]   # The card we want to insert into the sorted part
        j = i - 1      # Start comparing with the element right before it
        
        # Move elements of the sorted part that are greater than the key
        # one position ahead to make room for the key.
        while j >= 0 and arr[j] > key:
            arr[j + 1] = arr[j]  # Shift element to the right
            j -= 1               # Move backwards
            
        # Place the key into its correct sorted position
        arr[j + 1] = key
        
    print("Sorted array:", *arr)

⚡ Quick Sort

Divide & Conquer: Choose a pivot, partition the array so elements smaller than pivot go left and larger go right. Recursively sort both halves.

Key Property: After partitioning, the pivot is in its final sorted position.

⚠️ Worst Case
Quick Sort degrades to O(n²) when the pivot is always the smallest or largest element (e.g., already sorted array with first element as pivot). Use randomized pivot or median-of-three to avoid this.
#include <iostream>
#include <vector>
using namespace std;

// This function chooses a pivot and moves smaller elements to the left, larger to the right
int partition(vector<int>& arr, int low, int high) {
    int pivot = arr[high]; // We choose the last element as our pivot
    int i = low - 1;       // 'i' keeps track of where the smaller elements should go
    
    for (int j = low; j < high; j++) {
        // If the current element is smaller than or equal to our pivot
        if (arr[j] <= pivot) {
            i++; // Move our "small element boundary" forward
            swap(arr[i], arr[j]); // Swap the smaller element into the left side
        }
    }
    // Finally, place the pivot exactly in the middle (between small and large elements)
    swap(arr[i + 1], arr[high]);
    return i + 1; // Return the pivot's final position
}

void quickSort(vector<int>& arr, int low, int high) {
    // If the subarray has more than one element
    if (low < high) {
        // Partition the array and get the pivot index
        int pi = partition(arr, low, high);
        
        // Recursively sort the left part and the right part
        quickSort(arr, low, pi - 1);
        quickSort(arr, pi + 1, high);
    }
}

int main() {
    int n;
    cout << "Enter number of elements: ";
    cin >> n;
    
    vector<int> arr(n);
    cout << "Enter " << n << " elements: ";
    for (int i = 0; i < n; i++) {
        cin >> arr[i];
    }
    
    quickSort(arr, 0, n - 1);
    
    cout << "Sorted array: ";
    for (int i = 0; i < n; i++) {
        cout << arr[i] << " ";
    }
    cout << endl;
    
    return 0;
}
# This function chooses a pivot and moves smaller elements to the left, larger to the right
def partition(arr, low, high):
    pivot = arr[high]  # We choose the last element as our pivot
    i = low - 1        # 'i' keeps track of where the smaller elements should go
    
    for j in range(low, high):
        # If the current element is smaller than or equal to our pivot
        if arr[j] <= pivot:
            i += 1     # Move our "small element boundary" forward
            arr[i], arr[j] = arr[j], arr[i]  # Swap the smaller element into the left side
            
    # Finally, place the pivot exactly in the middle (between small and large elements)
    arr[i + 1], arr[high] = arr[high], arr[i + 1]
    return i + 1       # Return the pivot's final position

def quick_sort(arr, low, high):
    # If the subarray has more than one element
    if low < high:
        # Partition the array and get the pivot index
        pi = partition(arr, low, high)
        
        # Recursively sort the left part and the right part
        quick_sort(arr, low, pi - 1)
        quick_sort(arr, pi + 1, high)

if __name__ == "__main__":
    n = int(input("Enter number of elements: "))
    print(f"Enter {n} elements separated by space: ")
    arr = list(map(int, input().split()))
    
    quick_sort(arr, 0, len(arr) - 1)
    print("Sorted array:", *arr)

🔀 Merge Sort

Divide & Conquer: Split the array in half, recursively sort each half, then merge the two sorted halves.

Key Advantage: Always O(n log n) — no worst case degradation! Stable sort. Ideal for linked lists (no extra space needed for merging).

#include <iostream>
#include <vector>
using namespace std;

// This function takes two sorted halves and merges them into a single sorted array
void merge(vector<int>& arr, int l, int m, int r) {
    // Make temporary copies of the left and right halves
    vector<int> left(arr.begin() + l, arr.begin() + m + 1);
    vector<int> right(arr.begin() + m + 1, arr.begin() + r + 1);
    
    int i = 0; // Pointer for left half
    int j = 0; // Pointer for right half
    int k = l; // Pointer for the main array
    
    // Compare elements from both halves and pick the smaller one
    while (i < left.size() && j < right.size()) {
        if (left[i] <= right[j]) arr[k++] = left[i++];
        else arr[k++] = right[j++];
    }
    
    // If there are any remaining elements in the left half, copy them over
    while (i < left.size()) arr[k++] = left[i++];
    
    // If there are any remaining elements in the right half, copy them over
    while (j < right.size()) arr[k++] = right[j++];
}

void mergeSort(vector<int>& arr, int l, int r) {
    // If we have at least 2 elements
    if (l < r) {
        // Find the middle point to divide the array into two halves
        int m = l + (r - l) / 2;
        
        // Recursively sort the first and second halves
        mergeSort(arr, l, m);
        mergeSort(arr, m + 1, r);
        
        // Merge the two sorted halves back together
        merge(arr, l, m, r);
    }
}

int main() {
    int n;
    cout << "Enter number of elements: ";
    cin >> n;
    
    vector<int> arr(n);
    cout << "Enter " << n << " elements: ";
    for (int i = 0; i < n; i++) {
        cin >> arr[i];
    }
    
    mergeSort(arr, 0, n - 1);
    
    cout << "Sorted array: ";
    for (int i = 0; i < n; i++) {
        cout << arr[i] << " ";
    }
    cout << endl;
    
    return 0;
}
def merge_sort(arr):
    # A list of 0 or 1 element is already sorted!
    if len(arr) <= 1:
        return arr
    
    # Find the middle point to divide the array into two halves
    mid = len(arr) // 2
    
    # Recursively sort the first and second halves
    left = merge_sort(arr[:mid])
    right = merge_sort(arr[mid:])
    
    # Merge the two sorted halves back together into 'result'
    result = []
    i = j = 0
    
    # Compare elements from both halves and pick the smaller one
    while i < len(left) and j < len(right):
        if left[i] <= right[j]:
            result.append(left[i])
            i += 1
        else:
            result.append(right[j])
            j += 1
            
    # If there are any remaining elements in the left half, add them
    result.extend(left[i:])
    
    # If there are any remaining elements in the right half, add them
    result.extend(right[j:])
    
    return result

if __name__ == "__main__":
    n = int(input("Enter number of elements: "))
    print(f"Enter {n} elements separated by space: ")
    arr = list(map(int, input().split()))
    
    sorted_arr = merge_sort(arr)
    print("Sorted array:", *sorted_arr)

🔢 Counting Sort

Non-comparison sort: Count occurrences of each value, then use cumulative counts to place elements directly into their sorted position.

When to use: When the range of values (k) is not significantly larger than n. Perfect for sorting characters, grades, small integers.

#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

void countingSort(vector<int>& arr) {
    if (arr.empty()) return;
    
    // Find the largest and smallest numbers to know our range
    int maxVal = *max_element(arr.begin(), arr.end());
    int minVal = *min_element(arr.begin(), arr.end());
    int range = maxVal - minVal + 1;
    
    // count array stores how many times each number appears
    // output array will store our final sorted result
    vector<int> count(range, 0), output(arr.size());
    
    // Step 1: Count how many times each number appears in the array
    for (int x : arr) count[x - minVal]++;
    
    // Step 2: Cumulative count. This tells us the exact position 
    // of each element in the output array.
    for (int i = 1; i < range; i++) count[i] += count[i-1];
    
    // Step 3: Build the output array. We go backwards to keep the sort 'stable'
    // (meaning identical numbers keep their original relative order)
    for (int i = arr.size() - 1; i >= 0; i--) {
        output[count[arr[i] - minVal] - 1] = arr[i];
        count[arr[i] - minVal]--;
    }
    
    // Copy the sorted elements back into the original array
    arr = output;
}

int main() {
    int n;
    cout << "Enter number of elements: ";
    cin >> n;
    
    vector<int> arr(n);
    cout << "Enter " << n << " elements: ";
    for (int i = 0; i < n; i++) {
        cin >> arr[i];
    }
    
    countingSort(arr);
    
    cout << "Sorted array: ";
    for (int i = 0; i < n; i++) {
        cout << arr[i] << " ";
    }
    cout << endl;
    
    return 0;
}
def counting_sort(arr):
    if not arr:
        return arr
    
    # Find the largest and smallest numbers to know our range
    min_val, max_val = min(arr), max(arr)
    range_size = max_val - min_val + 1
    
    # count array stores how many times each number appears
    # output array will store our final sorted result
    count = [0] * range_size
    output = [0] * len(arr)
    
    # Step 1: Count how many times each number appears in the array
    for x in arr:
        count[x - min_val] += 1
    
    # Step 2: Cumulative count. This tells us the exact position 
    # of each element in the output array.
    for i in range(1, range_size):
        count[i] += count[i - 1]
    
    # Step 3: Build the output array. We go backwards to keep the sort 'stable'
    # (meaning identical numbers keep their original relative order)
    for i in range(len(arr) - 1, -1, -1):
        output[count[arr[i] - min_val] - 1] = arr[i]
        count[arr[i] - min_val] -= 1
    
    return output

if __name__ == "__main__":
    n = int(input("Enter number of elements: "))
    print(f"Enter {n} elements separated by space: ")
    arr = list(map(int, input().split()))
    
    sorted_arr = counting_sort(arr)
    print("Sorted array:", *sorted_arr)

📝 Teaching Notes

Suggested Teaching Sequence
  1. Show the comparison table first — give students the big picture (~10 min)
  2. Demo Bubble Sort with the visualizer — emphasize the optimization flag (~15 min)
  3. Explain Quick Sort's partition step on whiteboard before code (~20 min)
  4. Use the interactive sorting visualizer to compare all algorithms (~15 min)
  5. Assign 2 Easy + 1 Medium problem for lab, remaining for homework
Common Student Mistakes
  • Confusing stable vs. unstable sort — ask students to explain with examples
  • Off-by-one errors in Quick Sort partition boundaries
  • Forgetting the 'swapped' optimization in Bubble Sort
  • Not understanding why Merge Sort needs O(n) extra space
Discussion Questions
  • Why is Quick Sort generally faster than Merge Sort in practice despite same O(n log n)?
  • When would you prefer Insertion Sort over Quick Sort?
  • Is Counting Sort always better than comparison-based sorts?
  • What makes a sorting algorithm 'stable' and when does it matter?
