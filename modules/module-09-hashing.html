#️⃣ Hashing Fundamentals

A hash map (dictionary) provides average O(1) insertion, deletion, and lookup by mapping keys to values via a hash function.

OperationAverageWorst
InsertO(1)O(n)
LookupO(1)O(n)
DeleteO(1)O(n)
💡 Common Hashing Patterns
Frequency counting: Count occurrences of each element
Two Sum pattern: Store complement in a map
Grouping: Use sorted string or character count as key
Existence check: Use a HashSet for O(1) contains
