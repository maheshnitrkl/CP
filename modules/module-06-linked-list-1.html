📖 Theory

🔗 Linked List Basics

A linked list is a linear data structure where elements are stored in nodes, and each node points to the next. Unlike arrays, linked lists don't require contiguous memory.

OperationArrayLinked List
Access by indexO(1)O(n)
Insert at headO(n)O(1)
Insert at tailO(1)*O(n) / O(1)
DeleteO(n)O(1) if node known
SearchO(n)O(n)
💡 Key Technique: Two Pointers
Many linked list problems use slow/fast pointers. Slow moves 1 step, fast moves 2 steps. When fast reaches the end, slow is at the middle. Also used for cycle detection (Floyd's algorithm).
