📖 Theory

🌳 Binary Tree Fundamentals

A binary tree is a hierarchical data structure where each node has at most two children (left and right).

Traversal Orders:

  • Preorder (NLR): Visit node → left subtree → right subtree
  • Inorder (LNR): Left subtree → visit node → right subtree (gives sorted order for BST)
  • Postorder (LRN): Left subtree → right subtree → visit node
  • Level Order (BFS): Level by level, left to right
PropertyFormula
Max nodes at level L2^L
Max nodes in tree of height H2^(H+1) - 1
Min height for N nodes⌊log₂(N)⌋
💡 Recursion Pattern
Most tree problems follow this pattern: Base case (null node) → Recurse on left and right → Combine results. Think about what information each node needs from its children.
