Linked List Cycle II

Medium 🔗 LeetCode

📜 Problem Statement

Given a linked list, return the node where the cycle begins. If no cycle, return null.

🔢 Mathematical Foundation

Mathematical Proof

Let the distance from head to cycle start = a, cycle start to meeting point = b, and remaining cycle = c.

Slow travels: a + b. Fast travels: a + b + (b + c) = a + 2b + c.

Since fast = 2 × slow: a + 2b + c = 2(a + b) → a = c

So the distance from head to cycle start equals the distance from meeting point to cycle start! Move one pointer from head and one from meeting point, both at speed 1. They meet at the cycle start!

💡 Approach Progression: Brute Force → Optimal

Approach 1
Hash Set

Track visited nodes. The first repeated node is the cycle start.

function detectCycle(head):
  visited = Set()
  while head:
    if head in visited: return head
    visited.add(head); head = head.next
  return null
Time: O(n) Space: O(n)
Pros
  • Simple
Cons
  • O(n) space
Why do we need a better approach?
Floyd's algorithm extended with the a = c proof finds the start in O(1) space.
Approach 2
Floyd's + Math (a = c)

Find meeting point with slow/fast. Then move one pointer from head, one from meeting point, both at speed 1. They meet at cycle start.

function detectCycle(head):
  slow = fast = head
  while fast AND fast.next:
    slow = slow.next; fast = fast.next.next
    if slow == fast: // cycle detected
      entry = head
      while entry != slow:
        entry = entry.next; slow = slow.next
      return entry
  return null
Time: O(n) Space: O(1)
Pros
  • O(1) space
  • Beautiful mathematical proof
  • Industry-standard algorithm
Cons
  • The proof of a = c is non-obvious

💻 Optimal Implementation

class Solution {
public:
    ListNode *detectCycle(ListNode *head) {
        ListNode *slow = head, *fast = head;
        while (fast && fast->next) {
            slow = slow->next; fast = fast->next->next;
            if (slow == fast) {
                ListNode *entry = head;
                while (entry != slow) { entry = entry->next; slow = slow->next; }
                return entry;
            }
        }
        return nullptr;
    }
};
