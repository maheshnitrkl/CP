Sort Colors

Medium 🔗 LeetCode

📜 Problem Statement

Given an array with values 0, 1, and 2 (representing red, white, blue), sort them in-place so that same colors are adjacent in order 0, 1, 2.

Example: [2,0,2,1,1,0] → [0,0,1,1,2,2]

Constraint: One-pass algorithm using O(1) extra space.

🔢 Mathematical Foundation

Dutch National Flag Problem (Dijkstra, 1976)

Three pointers partition the array into four regions: [0..low-1] = 0s, [low..mid-1] = 1s, [mid..high] = unprocessed, [high+1..n-1] = 2s.

💡 Approach Progression: Brute Force → Optimal

Approach 1
Count & Overwrite

Count occurrences of 0, 1, 2. Then overwrite the array with the correct counts.

function sortColors(nums):
  c0 = count(0), c1 = count(1), c2 = count(2)
  fill nums with c0 zeros, c1 ones, c2 twos
Time: O(n) Space: O(1)
Pros
  • Simple two-pass solution
  • Easy to understand
Cons
  • Two passes through the array
  • Not a true in-place sorting algorithm
Why do we need a better approach?
Can we do it in a single pass? The Dutch National Flag algorithm achieves this.
Approach 2
Dutch National Flag (3-Way Partition)

Use three pointers: low, mid, high. Process mid pointer: if 0, swap with low and advance both. If 1, just advance mid. If 2, swap with high and decrement high.

function sortColors(nums):
  low = 0, mid = 0, high = n-1
  while mid ≤ high:
    if nums[mid] == 0:
      swap(nums[low], nums[mid])
      low++; mid++
    elif nums[mid] == 1:
      mid++ // 1 is in correct region
    else: // nums[mid] == 2
      swap(nums[mid], nums[high])
      high-- // don't advance mid! swapped value unknown
Time: O(n) Space: O(1)
Pros
  • Single pass
  • True in-place
  • Classic algorithm by Dijkstra
Cons
  • Three-pointer logic can be tricky

💻 Optimal Implementation

class Solution {
public:
    void sortColors(vector<int>& nums) {
        int low = 0, mid = 0, high = nums.size() - 1;
        while (mid <= high) {
            if (nums[mid] == 0) swap(nums[low++], nums[mid++]);
            else if (nums[mid] == 1) mid++;
            else swap(nums[mid], nums[high--]);
        }
    }
};
class Solution:
    def sortColors(self, nums):
        low, mid, high = 0, 0, len(nums) - 1
        while mid <= high:
            if nums[mid] == 0:
                nums[low], nums[mid] = nums[mid], nums[low]
                low += 1; mid += 1
            elif nums[mid] == 1:
                mid += 1
            else:
                nums[mid], nums[high] = nums[high], nums[mid]
                high -= 1
