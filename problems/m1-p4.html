Total Chocolates II

Easy 🔗 GFG

📝 Problem Statement

Find the optimal solution for this problem.

🔢 Mathematical Foundation

Paired Divisors up to √n

If a × b = n, then at least one of the factors must be less than or equal to √n.

Thus, we can find all divisors by iterating i up to √n. If i divides n, both i and n/i are divisors.

💡 Approach Progression: Brute Force → Optimal

Approach 1
Iterate to n

Iterate i from 1 to n. If n % i == 0, add i to the total sum.

Time: O(n) Space: O(1)
Pros
  • Simple and easy to write
Cons
  • Too slow if n is up to 10^9 or higher
Why do we need a better approach?
Divisors always appear in pairs. We only need to check up to √n.
Approach 2
Iterate to √n

Iterate i from 1 to √n. If i divides n, add both i and n/i to the sum. Handle perfect squares to avoid double counting.

for i = 1 to √n:
  if n % i == 0:
    total += i
    if i != n / i:
      total += n / i
Time: O(√n) Space: O(1)
Pros
  • Optimal time complexity
Cons
  • Requires handling the perfect square edge case

💻 Optimal Implementation

class Solution {
public:
    long long totalChocolates(int n) {
        long long total = 0;
        for (int i = 1; i * i <= n; i++) {
            if (n % i == 0) {
                total += i;
                if (i != n / i) {
                    total += n / i;
                }
            }
        }
        return total;
    }
};
class Solution:
    def totalChocolates(self, n):
        total = 0
        i = 1
        while i * i <= n:
            if n % i == 0:
                total += i
                if i != n // i:
                    total += n // i
            i += 1
        return total
