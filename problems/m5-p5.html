Palindrome Partitioning

Medium 🔗 LeetCode

📜 Problem Statement

Given a string s, partition it such that every substring is a palindrome. Return all possible palindrome partitionings.

Example: "aab" → [["a","a","b"],["aa","b"]]

🔢 Mathematical Foundation

A string of length n has at most 2n-1 ways to partition

Between each pair of adjacent characters, we can either cut or not. That's n-1 positions with 2 choices each = 2n-1 partitions. We prune those where any part isn't a palindrome.

💡 Approach Progression: Brute Force → Optimal

Approach 1
Generate All Partitions + Filter

Generate all 2n-1 partitions, then check each one for palindromicity.

function partition(s):
  allPartitions = generateAll(s)
  result = []
  for p in allPartitions:
    if allPalindromes(p):
      result.append(p)
  return result
Time: O(n × 2n) Space: O(n × 2n)
Pros
  • Conceptually simple
Cons
  • Generates many invalid partitions
  • No pruning
Why do we need a better approach?
We're generating invalid partitions and checking afterwards. We should prune early: only recurse if the current prefix is a palindrome.
Approach 2
Backtracking with Early Palindrome Check

At each step, try all possible prefixes. Only recurse if the prefix is a palindrome. This prunes entire subtrees early.

function partition(s):
  result = []
  function backtrack(start, current):
    if start == len(s):
      result.append(copy(current)); return
    for end = start to len(s)-1:
      substr = s[start..end]
      if isPalindrome(substr): // PRUNE here!
        current.append(substr)
        backtrack(end + 1, current)
        current.pop()
  backtrack(0, []); return result
Time: O(n × 2n) worst case Space: O(n)
Pros
  • Early pruning avoids invalid partitions
  • Clean recursive structure
  • Can be further optimized with DP palindrome precomputation
Cons
  • Palindrome check is O(n) per call

💻 Optimal Implementation

class Solution {
public:
    vector<vector<string>> partition(string s) {
        vector<vector<string>> result;
        vector<string> current;
        backtrack(s, 0, current, result);
        return result;
    }
    bool isPalin(string& s, int l, int r) {
        while (l < r) if (s[l++] != s[r--]) return false;
        return true;
    }
    void backtrack(string& s, int start, vector<string>& cur,
                   vector<vector<string>>& res) {
        if (start == s.size()) { res.push_back(cur); return; }
        for (int end = start; end < s.size(); end++) {
            if (isPalin(s, start, end)) {
                cur.push_back(s.substr(start, end - start + 1));
                backtrack(s, end + 1, cur, res);
                cur.pop_back();
            }
        }
    }
};
class Solution:
    def partition(self, s):
        result = []
        def backtrack(start, current):
            if start == len(s):
                result.append(current[:]); return
            for end in range(start, len(s)):
                substr = s[start:end+1]
                if substr == substr[::-1]:  # palindrome check
                    current.append(substr)
                    backtrack(end + 1, current)
                    current.pop()
        backtrack(0, [])
        return result
