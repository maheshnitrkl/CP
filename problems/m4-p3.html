Merge Intervals

Medium 🔗 LeetCode

📜 Problem Statement

Given an array of intervals [[start, end]], merge all overlapping intervals.

Example: [[1,3],[2,6],[8,10],[15,18]] → [[1,6],[8,10],[15,18]]

🔢 Mathematical Foundation

Two intervals [a,b] and [c,d] overlap if and only if a ≤ d AND c ≤ b

After sorting by start time, we only need to check if the current interval overlaps with the last merged one (current.start ≤ last.end).

💡 Approach Progression: Brute Force → Optimal

Approach 1
Check All Pairs

Compare every pair of intervals. If they overlap, merge them. Repeat until no more merges possible.

repeat:
  merged = false
  for each pair (i, j):
    if overlaps(i, j):
      merge(i, j); merged = true
until not merged
Time: O(n2) or worse Space: O(n)
Pros
  • No sorting needed
Cons
  • O(n^2) pair comparisons
  • May need multiple passes
  • Complex merge logic
Why do we need a better approach?
Sorting by start time means we only need to compare adjacent intervals. One pass after sorting!
Approach 2
Sort + Linear Merge

Sort intervals by start time. Then iterate: if current interval overlaps with the last merged one, extend the end. Otherwise, start a new merged interval.

function merge(intervals):
  sort intervals by start time
  result = [intervals[0]]
  for i = 1 to n-1:
    if intervals[i].start ≤ result.last.end:
      result.last.end = max(result.last.end, intervals[i].end)
    else:
      result.append(intervals[i])
  return result
Time: O(n log n) Space: O(n)
Pros
  • O(n log n) dominated by sorting
  • Single pass after sorting
  • Clean and elegant
Cons
  • Sorting modifies order

💻 Optimal Implementation

class Solution {
public:
    vector<vector<int>> merge(vector<vector<int>>& intervals) {
        sort(intervals.begin(), intervals.end());
        vector<vector<int>> result = {intervals[0]};
        for (int i = 1; i < intervals.size(); i++) {
            if (intervals[i][0] <= result.back()[1])
                result.back()[1] = max(result.back()[1], intervals[i][1]);
            else
                result.push_back(intervals[i]);
        }
        return result;
    }
};
class Solution:
    def merge(self, intervals):
        intervals.sort()
        result = [intervals[0]]
        for start, end in intervals[1:]:
            if start <= result[-1][1]:
                result[-1][1] = max(result[-1][1], end)
            else:
                result.append([start, end])
        return result
