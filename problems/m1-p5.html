Count Primes

Medium 🔗 LeetCode

📜 Problem Statement

Count the number of prime numbers strictly less than n.

🔢 Mathematical Foundation

Sieve of Eratosthenes

To find all primes up to n, assume all numbers are prime. Starting from 2, if a number is prime, mark all of its multiples starting from its square as not prime.

💡 Approach Progression: Brute Force → Optimal

Approach 1
Check Each Number

For each number up to n, call isPrime(x).

function countPrimes(n):
  count = 0
  for i = 2 to n-1:
    if isPrime(i): count++
  return count
Time: O(n√n) Space: O(1)
Pros
  • Easy to write if isPrime exists
Cons
  • Too slow for large ranges
Why do we need a better approach?
We recalculate properties of numbers repeatedly. The Sieve eliminates composites efficiently.
Approach 2
Sieve of Eratosthenes

Array of booleans. Mark multiples as false.

function countPrimes(n):
  isPrime = array of true
  isPrime[0] = isPrime[1] = false
  for i = 2 to √n:
    if isPrime[i]:
      for j = i*i to n step i:
        isPrime[j] = false
  return count of true values
Time: O(n log log n) Space: O(n)
Pros
  • Extremely fast for range queries
  • Beautiful algorithm
Cons
  • Requires O(n) memory

💻 Optimal Implementation

class Solution {
public:
    int countPrimes(int n) {
        if (n <= 2) return 0;
        vector<bool> isPrime(n, true);
        isPrime[0] = isPrime[1] = false;
        for (int i = 2; i * i < n; i++) {
            if (isPrime[i]) {
                for (int j = i * i; j < n; j += i) {
                    isPrime[j] = false;
                }
            }
        }
        return count(isPrime.begin(), isPrime.end(), true);
    }
};
class Solution:
    def countPrimes(self, n: int) -> int:
        if n <= 2: return 0
        is_prime = [True] * n
        is_prime[0] = is_prime[1] = False
        for i in range(2, int(n**0.5) + 1):
            if is_prime[i]:
                for j in range(i*i, n, i):
                    is_prime[j] = False
        return sum(is_prime)
