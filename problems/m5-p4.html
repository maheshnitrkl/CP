Subsets II

Medium 🔗 LeetCode

📜 Problem Statement

Given an integer array nums that may contain duplicates, return all possible unique subsets.

Example: [1,2,2] → [[], [1], [1,2], [1,2,2], [2], [2,2]]

🔢 Mathematical Foundation

Without duplicates: 2n subsets. With duplicates, some subsets are identical. We must avoid generating them.

Rule: At each recursion level, skip nums[i] if nums[i] == nums[i-1] and i > start

This ensures that among consecutive identical elements, we only use the first one to begin a new branch at each level.

💡 Approach Progression: Brute Force → Optimal

Approach 1
Generate All + Deduplicate

Generate all 2n subsets, sort each, put in a set to remove duplicates.

function subsetsWithDup(nums):
  allSubsets = generateAllSubsets(nums)
  uniqueSet = Set()
  for s in allSubsets:
    uniqueSet.add(sort(s))
  return list(uniqueSet)
Time: O(n × 2n × log n) Space: O(n × 2n)
Pros
  • Simple to understand
Cons
  • Sorting each subset is expensive
  • Set comparison is expensive
  • Generates then throws away duplicates
Why do we need a better approach?
Just like Permutations II, we should avoid generating duplicates rather than filtering them.
Approach 2
Sort + Skip at Same Level

Sort the array. During backtracking, if nums[i] == nums[i-1] and i > start, skip it (prevents picking the same element twice at the same decision level).

function subsetsWithDup(nums):
  sort(nums)
  result = []
  function backtrack(start, current):
    result.append(copy(current))
    for i = start to n-1:
      if i > start AND nums[i] == nums[i-1]: continue
      current.append(nums[i])
      backtrack(i + 1, current)
      current.pop()
  backtrack(0, []); return result
Time: O(n × 2n) Space: O(n)
Pros
  • No duplicate subsets generated
  • One extra line vs Subsets I
  • Efficient pruning
Cons
  • Must sort first

💻 Optimal Implementation

class Solution {
public:
    vector<vector<int>> subsetsWithDup(vector<int>& nums) {
        sort(nums.begin(), nums.end());
        vector<vector<int>> result;
        vector<int> current;
        backtrack(nums, 0, current, result);
        return result;
    }
    void backtrack(vector<int>& nums, int start,
                   vector<int>& cur, vector<vector<int>>& res) {
        res.push_back(cur);
        for (int i = start; i < nums.size(); i++) {
            if (i > start && nums[i] == nums[i-1]) continue; // skip dup
            cur.push_back(nums[i]);
            backtrack(nums, i + 1, cur, res);
            cur.pop_back();
        }
    }
};
class Solution:
    def subsetsWithDup(self, nums):
        nums.sort()
        result = []
        def backtrack(start, current):
            result.append(current[:])
            for i in range(start, len(nums)):
                if i > start and nums[i] == nums[i-1]: continue
                current.append(nums[i])
                backtrack(i + 1, current)
                current.pop()
        backtrack(0, [])
        return result
