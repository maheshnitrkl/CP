Minimum Moves to Equal Array Elements

Medium 🔗 LeetCode

📜 Problem Statement

Given an integer array nums, return the minimum number of moves to make all elements equal, where each move increments n-1 elements by 1.

Example: [1,2,3] → 3 moves. [1,2,3] → [2,3,3] → [3,4,3] → [4,4,4]

🔢 Mathematical Foundation

Incrementing (n-1) elements by 1 = Decrementing 1 element by 1

Instead of thinking about raising n-1 elements, think about lowering 1 element. The minimum moves = total steps to bring everything down to the minimum element.

moves = Σ(nums[i] - min(nums))

💡 Approach Progression: Brute Force → Optimal

Approach 1
Simulation

Actually simulate the moves: each step, increment all except the maximum. Stop when all equal.

while not allEqual(nums):
  maxIdx = findMax(nums)
  for i = 0 to n-1:
    if i != maxIdx: nums[i] += 1
  moves += 1
Time: O(moves × n) Space: O(1)
Pros
  • Direct simulation of the problem
Cons
  • Extremely slow for large differences
  • Could take millions of iterations
Why do we need a better approach?
The simulation is painfully slow. The math insight: incrementing n-1 is equivalent to decrementing 1. This gives us a closed-form solution.
Approach 2
Mathematical Formula

Incrementing n-1 elements by 1 is equivalent to decrementing 1 element by 1. So, the answer is the sum of differences from the minimum element.

function minMoves(nums):
  minVal = min(nums)
  moves = 0
  for x in nums:
    moves += x - minVal
  return moves

// Equivalent: sum(nums) - n * min(nums)
Time: O(n) Space: O(1)
Pros
  • O(n) single pass
  • Elegant mathematical insight
  • No simulation needed
Cons
  • Requires the key insight that +1 to (n-1) = -1 to 1

💻 Optimal Implementation

class Solution {
public:
    int minMoves(vector<int>& nums) {
        int minVal = *min_element(nums.begin(), nums.end());
        int moves = 0;
        for (int x : nums) moves += x - minVal;
        return moves;
    }
};
class Solution:
    def minMoves(self, nums):
        min_val = min(nums)
        return sum(x - min_val for x in nums)
