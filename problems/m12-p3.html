Keys and Rooms

Medium 🔗 Problem Link

📜 Problem Statement

There are n rooms labeled from 0 to n - 1 and all the rooms are locked except for room 0. Your goal is to visit all the rooms. When you visit a room, you may find a set of distinct keys in it that can unlock other rooms.

📚 Logic & Core Concept

You start in Room 0, which is unlocked. Inside each room, you might find keys to other rooms.

This is a classic graph traversal problem where rooms are nodes and keys are directed edges.

rooms = [[1], [2], [3], []] means Room 0 has a key to Room 1, Room 1 has a key to Room 2, etc.

To see if we can visit all rooms, we start at Room 0, follow the keys to open new rooms, and keep track of which rooms we've visited. If the total number of visited rooms equals n, we win.

💡 Approach Progression: DFS → BFS

Approach 1
Depth-First Search (DFS)

We can use a simple recursive DFS. When we enter a room, we mark it as visited, pick up all the keys, and immediately try to enter those rooms if we haven't already. A global counter tracks how many rooms we've opened.

function canVisitAllRooms(rooms):
  n = len(rooms)
  visited = array of false (size n)
  visited[0] = true
  count = 1
  
  function dfs(room):
    for key in rooms[room]:
      if not visited[key]:
        visited[key] = true
        count += 1
        dfs(key)
  
  dfs(0)
  return count == n
Time: O(N + E) Space: O(N)
Pros
  • Short and elegant code.
  • No need to build an adjacency list (it's already given as `rooms`).
Cons
  • Recursion stack depth could reach N.
Alternative
If you prefer avoiding deep recursion, a standard Breadth-First Search (BFS) works just as well.
Approach 2
Breadth-First Search (BFS)

We use a queue. Start with Room 0 in the queue. Pop a room, grab its keys, and for every key to an unvisited room, mark it visited and push it to the queue.

function canVisitAllRoomsBFS(rooms):
  visited = set([0])
  queue = [0]
  
  while queue is not empty:
    curr = queue.dequeue()
    for key in rooms[curr]:
      if key not in visited:
        visited.add(key)
        queue.enqueue(key)
  
  return len(visited) == len(rooms)
Time: O(N + E) Space: O(N)

💻 Optimal Implementation

class Solution {
    int count = 0;
    
    void dfs(int room, vector<vector<int>>& rooms, vector<bool>& visited) {
        visited[room] = true;
        count++;
        
        for (int key : rooms[room]) {
            if (!visited[key]) {
                dfs(key, rooms, visited);
            }
        }
    }
    
public:
    bool canVisitAllRooms(vector<vector<int>>& rooms) {
        int n = rooms.size();
        vector<bool> visited(n, false);
        
        dfs(0, rooms, visited);
        
        return count == n;
    }
};
class Solution:
    def canVisitAllRooms(self, rooms: List[List[int]]) -> bool:
        n = len(rooms)
        visited = [False] * n
        self.count = 0
        
        def dfs(room):
            visited[room] = True
            self.count += 1
            for key in rooms[room]:
                if not visited[key]:
                    dfs(key)
                    
        dfs(0)
        return self.count == n
