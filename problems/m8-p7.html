Gas Station

Medium 🔗 LeetCode

📜 Problem Statement

Given circular arrays gas and cost, find the starting station to complete the circuit. Return -1 if impossible.

🔢 Mathematical Foundation

Greedy Property

1. If sum(gas) < sum(cost), impossible to complete circuit.
2. If you start at A and can't reach B, no station between A and B can reach B either. Set start = B + 1.

💡 Approach Progression: Brute Force → Optimal

Approach 1
Brute Force Simulation

Try starting at every single station `i`. For each station, simulate the journey around the circle. If the tank drops below 0, break and try the next station.

function canCompleteCircuit(gas, cost):
  n = len(gas)
  for start = 0 to n-1:
    tank = 0, steps = 0
    while steps < n:
      curr = (start + steps) % n
      tank += gas[curr] - cost[curr]
      if tank < 0: break
      steps += 1
    if steps == n: return start
  return -1
Time: O(n2) Space: O(1)
Pros
  • Easy to understand and implement
Cons
  • Time Limit Exceeded (TLE) on large inputs
Why optimize?
When we fail to reach station B from A, no station between A and B can reach B either! So we can safely skip all of them and try B+1.
Approach 2
Greedy O(n) Pass

Maintain total_tank and current_tank. If current_tank < 0, reset start to next station and current_tank to 0. If total_tank >= 0 at end, return start.

function canCompleteCircuit(gas, cost):
  total = 0, curr = 0, start = 0
  for i = 0 to n-1:
    diff = gas[i] - cost[i]
    total += diff; curr += diff
    if curr < 0:
      start = i + 1; curr = 0
  return total >= 0 ? start : -1
Time: O(n) Space: O(1)
Pros
  • Single pass
  • Brilliant greedy logic
Cons
  • Requires proving the greedy choice is valid

💻 Optimal Implementation

class Solution {
public:
    int canCompleteCircuit(vector<int>& gas, vector<int>& cost) {
        int total = 0, curr = 0, start = 0;
        for (int i = 0; i < gas.size(); i++) {
            int diff = gas[i] - cost[i];
            total += diff; curr += diff;
            if (curr < 0) { start = i + 1; curr = 0; }
        }
        return total >= 0 ? start : -1;
    }
};
