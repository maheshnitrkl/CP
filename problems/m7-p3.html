Palindrome Linked List

Easy 🔗 LeetCode

📜 Problem Statement

Determine if a singly linked list is a palindrome.

Example: 1→2→2→1 → true

💡 Approach Progression: Brute Force → Optimal

Approach 1
Copy to Array

Copy all values to an array, then use two pointers to check palindrome.

function isPalindrome(head):
  arr = toArray(head)
  return arr == reversed(arr)
Time: O(n) Space: O(n)
Pros
  • Very simple
Cons
  • O(n) extra space
Why do we need a better approach?
We can check in-place by reversing the second half.
Approach 2
Reverse Second Half In-Place

Find the middle (slow/fast). Reverse the second half. Compare both halves node by node.

function isPalindrome(head):
  mid = findMiddle(head) // slow/fast
  secondHalf = reverse(mid)
  while secondHalf:
    if head.val != secondHalf.val: return false
    head = head.next; secondHalf = secondHalf.next
  return true
Time: O(n) Space: O(1)
Pros
  • O(1) space
  • Combines two techniques: find middle + reverse
Cons
  • Modifies the list (can restore afterwards)

💻 Optimal Implementation

class Solution {
public:
    bool isPalindrome(ListNode* head) {
        ListNode *slow = head, *fast = head;
        while (fast && fast->next) { slow = slow->next; fast = fast->next->next; }
        ListNode *rev = nullptr, *curr = slow;
        while (curr) { ListNode *nxt = curr->next; curr->next = rev; rev = curr; curr = nxt; }
        while (rev) {
            if (head->val != rev->val) return false;
            head = head->next; rev = rev->next;
        }
        return true;
    }
};
class Solution:
    def isPalindrome(self, head):
        slow = fast = head
        while fast and fast.next:
            slow = slow.next; fast = fast.next.next
        prev = None
        while slow:
            nxt = slow.next; slow.next = prev; prev = slow; slow = nxt
        while prev:
            if head.val != prev.val: return False
            head = head.next; prev = prev.next
        return True
