Largest Prime Factor

Easy 🔗 GFG

📝 Problem Statement

Find the optimal solution for this problem.

🔢 Mathematical Foundation

Fundamental Theorem of Arithmetic

Every integer greater than 1 is either a prime itself or can be represented as the product of prime numbers.

By repeatedly dividing out the smallest prime factors (starting from 2, then odd numbers up to √n), the remaining number must either be 1 or the largest prime factor itself.

💡 Approach Progression: Brute Force → Optimal

Approach 1
Check Every Number

Iterate i from 2 to n. Check if i divides n AND if i is prime. Update maximum prime factor found.

Time: O(n × √n) Space: O(1)
Pros
  • Simple and intuitive
Cons
  • Extremely slow for large n
Why do we need a better approach?
We don't need to check all numbers up to n. Factors appear in pairs up to √n.
Approach 2
Iterate to √n

Iterate i up to √n to find factors. If i is a factor, check if i or n/i is prime.

Time: O(√n × √i) Space: O(1)
Pros
  • Much faster
Cons
  • Still does redundant primality checks
Why do we need a better approach?
Instead of finding factors and checking if they are prime, we can continuously divide n by its smallest prime factors to extract them dynamically.
Approach 3
Prime Factorization

Repeatedly divide out the smallest prime factor (starting with 2, then odd numbers up to √n). What remains is the largest prime factor.

while n % 2 == 0: max_prime = 2, n = n / 2
for i = 3 to √n step 2:
  while n % i == 0:
    max_prime = i, n = n / i
if n > 1: max_prime = n
Time: O(√n) Space: O(1)
Pros
  • Optimal time complexity
Cons
  • Requires careful while loops

💻 Optimal Implementation

class Solution {
public:
    long long largestPrimeFactor(int n) {
        long long maxPrime = -1;
        while (n % 2 == 0) {
            maxPrime = 2;
            n /= 2;
        }
        for (long long i = 3; i * i <= n; i += 2) {
            while (n % i == 0) {
                maxPrime = i;
                n /= i;
            }
        }
        if (n > 1) maxPrime = n;
        return maxPrime;
    }
};
class Solution:
    def largestPrimeFactor(self, n):
        max_prime = -1
        while n % 2 == 0:
            max_prime = 2
            n //= 2
        i = 3
        while i * i <= n:
            while n % i == 0:
                max_prime = i
                n //= i
            i += 2
        if n > 1:
            max_prime = n
        return max_prime
