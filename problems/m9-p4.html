Sort Characters By Frequency

Medium 🔗 LeetCode

📜 Problem Statement

Sort a string in decreasing order based on the frequency of characters.

💡 Approach Progression: Brute Force → Optimal

Approach 1
Hash Map + Max-Heap

Count character frequencies using a Hash Map. Push each `(frequency, char)` into a Max-Heap. Then, pop from the heap to build the final string.

function frequencySort(s):
  freq = Counter(s)
  heap = MaxHeap()
  for char, count in freq:
    heap.push((count, char))
  res = ""
  while not heap.empty():
    count, char = heap.pop()
    res += char * count
  return res
Time: O(n log k) Space: O(k)
Pros
  • Naturally sorts dynamically
Cons
  • Heap overhead
Why optimize?
Since all we need is a full sort of the keys, a simple array sort might be more straightforward.
Approach 2
Hash Map + Array Sort

Count frequencies, put the keys in a list, sort the characters by their frequency descending, and rebuild the string.

function frequencySort(s):
  freq = Counter(s)
  chars = list(freq.keys())
  chars.sort(key=lambda c: freq[c], reverse=True)
  res = ""
  for c in chars:
    res += c * freq[c]
  return res
Time: O(n log n) Space: O(n)
Pros
  • Simple pipeline
Cons
  • Sorting is O(n log n) in worst case
Why optimize?
The maximum possible frequency is the length of the string `n`. We can use Bucket Sort to achieve true O(n) time!
Approach 3
Hash Map + Bucket Sort

Create an array of "buckets" where the index represents the frequency. Place each character in the bucket matching its frequency. Traverse the buckets backwards (highest frequency first) to build the result.

function frequencySort(s):
  freq = Counter(s)
  buckets = array of size len(s) + 1
  for char, count in freq:
    buckets[count].append(char)
  res = ""
  for count from len(s) downTo 1:
    for char in buckets[count]:
      res += char * count
  return res
Time: O(n) Space: O(n)
Pros
  • True linear time O(n)
Cons
  • Uses slightly more space for the buckets array

💻 Optimal Implementation

class Solution {
public:
    string frequencySort(string s) {
        unordered_map<char, int> freq;
        for (char c : s) freq[c]++;
        sort(s.begin(), s.end(), [&](char a, char b) {
            return freq[a] == freq[b] ? a < b : freq[a] > freq[b];
        });
        return s;
    }
};
from collections import Counter
class Solution:
    def frequencySort(self, s):
        c = Counter(s)
        return "".join(char * freq for char, freq in c.most_common())
