Add Two Numbers

Medium 🔗 LeetCode

📜 Problem Statement

Two non-empty linked lists represent two non-negative integers stored in reverse order. Add them and return the sum as a linked list.

Example: 2→4→3 + 5→6→4 → 7→0→8 (342 + 465 = 807)

💡 Approach Progression: Brute Force → Optimal

Approach 1
Convert to Numbers, Add, Convert Back

Convert both lists to integers, add them, convert the sum back to a list.

function addTwoNumbers(l1, l2):
  num1 = listToInt(l1)
  num2 = listToInt(l2)
  return intToList(num1 + num2)
Time: O(n + m) Space: O(max(n,m))
Pros
  • Conceptually simple
Cons
  • Integer overflow for very long numbers
  • Not using linked list structure
Why do we need a better approach?
This fails for very large numbers (100+ digits). We should add digit by digit with carry, like elementary school addition.
Approach 2
Digit-by-Digit with Carry

Add corresponding digits plus carry. Create a new node for each digit of the result. Continue until both lists are exhausted and carry is 0.

function addTwoNumbers(l1, l2):
  dummy = ListNode(0)
  curr = dummy, carry = 0
  while l1 OR l2 OR carry:
    sum = carry
    if l1: sum += l1.val; l1 = l1.next
    if l2: sum += l2.val; l2 = l2.next
    carry = sum / 10
    curr.next = ListNode(sum % 10)
    curr = curr.next
  return dummy.next
Time: O(max(n,m)) Space: O(max(n,m))
Pros
  • Handles arbitrary length numbers
  • No integer overflow
  • Elegant carry propagation
Cons
  • Creates new nodes for the result

💻 Optimal Implementation

class Solution {
public:
    ListNode* addTwoNumbers(ListNode* l1, ListNode* l2) {
        ListNode dummy(0), *curr = &dummy;
        int carry = 0;
        while (l1 || l2 || carry) {
            int sum = carry;
            if (l1) { sum += l1->val; l1 = l1->next; }
            if (l2) { sum += l2->val; l2 = l2->next; }
            carry = sum / 10;
            curr->next = new ListNode(sum % 10);
            curr = curr->next;
        }
        return dummy.next;
    }
};
class Solution:
    def addTwoNumbers(self, l1, l2):
        dummy = curr = ListNode(0)
        carry = 0
        while l1 or l2 or carry:
            s = carry
            if l1: s += l1.val; l1 = l1.next
            if l2: s += l2.val; l2 = l2.next
            carry, digit = divmod(s, 10)
            curr.next = ListNode(digit)
            curr = curr.next
        return dummy.next
