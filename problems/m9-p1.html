Find Common Elements Between Two Arrays

Easy 🔗 LeetCode

📜 Problem Statement

Return the number of elements in nums1 that exist in nums2, and vice versa.

💡 Approach Progression: Brute Force → Optimal

Approach 1
Nested Loops

For each element in nums1, search for it in nums2 using a loop. Repeat the process for nums2 searching in nums1.

function findIntersectionValues(nums1, nums2):
  ans1 = 0, ans2 = 0
  for x in nums1:
    if nums2.contains(x): ans1++
  for x in nums2:
    if nums1.contains(x): ans2++
  return [ans1, ans2]
Time: O(n × m) Space: O(1)
Pros
  • No extra space
  • Simple logic
Cons
  • Extremely slow for large arrays
Why optimize?
The inner `.contains(x)` does a linear scan every time. We can speed this up by sorting the arrays and using Binary Search.
Approach 2
Sort + Binary Search

Sort nums2. For each element in nums1, use binary search to check if it exists in nums2. Repeat for the other side.

function findIntersectionValues(nums1, nums2):
  nums1.sort(), nums2.sort()
  ans1 = sum(1 for x in nums1 if binarySearch(nums2, x))
  ans2 = sum(1 for x in nums2 if binarySearch(nums1, x))
  return [ans1, ans2]
Time: O(n log n + m log m) Space: O(1) or O(log n)
Pros
  • Much faster than O(n*m)
  • Minimal extra space
Cons
  • Mutates original arrays
Why optimize?
Binary Search still takes O(log n) per lookup. If we trade space for time, we can get O(1) lookups using Hash Sets.
Approach 3
Hash Sets

Store both arrays in Hash Sets for O(1) lookups.

function findIntersectionValues(nums1, nums2):
  s1 = Set(nums1), s2 = Set(nums2)
  ans1 = sum(1 for x in nums1 if x in s2)
  ans2 = sum(1 for x in nums2 if x in s1)
  return [ans1, ans2]
Time: O(n + m) Space: O(n + m)
Pros
  • Very fast O(1) lookups
Cons
  • Needs extra space for sets

💻 Optimal Implementation

class Solution {
public:
    vector<int> findIntersectionValues(vector<int>& nums1, vector<int>& nums2) {
        unordered_set<int> s1(nums1.begin(), nums1.end());
        unordered_set<int> s2(nums2.begin(), nums2.end());
        int c1 = 0, c2 = 0;
        for (int x : nums1) if (s2.count(x)) c1++;
        for (int x : nums2) if (s1.count(x)) c2++;
        return {c1, c2};
    }
};
class Solution:
    def findIntersectionValues(self, nums1, nums2):
