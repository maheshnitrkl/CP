Power of 2

Easy 🔗 LeetCode

📜 Problem Statement

Given an integer n, return true if it is a power of two. Otherwise, return false.

Example: n = 16 → true (16 = 24). n = 6 → false.

Constraints: -231 ≤ n ≤ 231 - 1

🔢 Mathematical Foundation

A power of 2 has exactly ONE set bit in binary.

20 = 1 (1), 21 = 2 (10), 22 = 4 (100), 23 = 8 (1000)...

If n has only 1 set bit, then n & (n-1) == 0, because (n-1) flips all bits below that single set bit.

Example: n=8 (1000), n-1=7 (0111). 1000 & 0111 = 0000. Power of 2!

💡 Approach Progression: Brute Force → Optimal

Approach 1
Repeated Division

Keep dividing n by 2. If we reach 1, it's a power of 2. If at any point n is odd (and not 1), it's not.

function isPowerOfTwo(n):
  if n ≤ 0: return false
  while n > 1:
    if n % 2 != 0: return false
    n = n / 2
  return true
Time: O(log n) Space: O(1)
Pros
  • Very intuitive
  • Uses basic division
Cons
  • Requires a loop
  • log(n) iterations for large numbers
Why do we need a better approach?
Division is expensive compared to a single bitwise operation. Can we check in O(1)?
Approach 2
Bit Trick: n & (n-1)

A power of 2 has exactly one set bit. n & (n-1) removes the lowest set bit. If only one bit is set, the result is 0. This is an O(1) check!

function isPowerOfTwo(n):
  return n > 0 AND (n & (n - 1)) == 0
Time: O(1) Space: O(1)
Pros
  • Single operation
  • Constant time regardless of input size
  • Elegant and efficient
Cons
  • Requires understanding of bitwise math

👀 Step-by-Step Walkthrough

Trace: Is 16 a power of 2?
n = 16 = 10000
n - 1 = 15 = 01111
n & (n-1) = 10000 & 01111 = 00000 = 0
Since n > 0 and result is 0 → True! 16 is a power of 2.
Trace: Is 6 a power of 2?
n = 6 = 110
n - 1 = 5 = 101
n & (n-1) = 110 & 101 = 100 = 4 (not 0!)
Since result is NOT 0 → False! 6 is not a power of 2.

💻 Optimal Implementation

class Solution {
public:
    bool isPowerOfTwo(int n) {
        return n > 0 && (n & (n - 1)) == 0;
    }
};
