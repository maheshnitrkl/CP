Search a 2D Matrix

Medium 🔗 LeetCode

📜 Problem Statement

Given an m x n matrix where each row is sorted and the first element of each row is greater than the last element of the previous row, search for a target value.

Example: matrix = [[1,3,5,7],[10,11,16,20],[23,30,34,60]], target = 3 → true

🔢 Mathematical Foundation

Index Mapping: row = idx / cols, col = idx % cols

The matrix is effectively a sorted 1D array of length m×n. We can binary search on this virtual 1D array using the index mapping formula.

💡 Approach Progression: Brute Force → Optimal

Approach 1
Scan Entire Matrix

Check every element in the matrix.

for each row in matrix:
  for each element in row:
    if element == target: return true
return false
Time: O(m × n) Space: O(1)
Pros
  • Dead simple
Cons
  • Ignores all sorting properties
Why do we need a better approach?
Both rows and the overall matrix are sorted. We can do much better.
Approach 2
Binary Search Per Row

First, binary search for the correct row (based on first/last element), then binary search within that row.

function searchMatrix(matrix, target):
  // Find correct row
  for each row:
    if row[0] ≤ target ≤ row[last]:
      return binarySearch(row, target)
  return false
Time: O(m + log n) Space: O(1)
Pros
  • Uses row sorting
Cons
  • Finding the row is still O(m)
Why do we need a better approach?
We can treat the entire matrix as a single sorted array and do a single binary search.
Approach 3
Single Binary Search on Virtual 1D Array

Treat the matrix as a 1D sorted array of length m×n. Use row = mid/n, col = mid%n to map indices.

function searchMatrix(matrix, target):
  m = rows, n = cols
  lo = 0, hi = m*n - 1
  while lo ≤ hi:
    mid = lo + (hi - lo) / 2
    val = matrix[mid / n][mid % n]
    if val == target: return true
    elif val < target: lo = mid + 1
    else: hi = mid - 1
  return false
Time: O(log(m × n)) Space: O(1)
Pros
  • Single O(log(mn)) binary search
  • Clean index mapping
  • Most efficient
Cons
  • Index math can be confusing

💻 Optimal Implementation

class Solution {
public:
    bool searchMatrix(vector<vector<int>>& matrix, int target) {
        int m = matrix.size(), n = matrix[0].size();
        int lo = 0, hi = m * n - 1;
        while (lo <= hi) {
            int mid = lo + (hi - lo) / 2;
            int val = matrix[mid / n][mid % n];
            if (val == target) return true;
            else if (val < target) lo = mid + 1;
            else hi = mid - 1;
        }
        return false;
    }
};
class Solution:
    def searchMatrix(self, matrix, target):
        m, n = len(matrix), len(matrix[0])
        lo, hi = 0, m * n - 1
        while lo <= hi:
            mid = (lo + hi) // 2
            val = matrix[mid // n][mid % n]
            if val == target: return True
            elif val < target: lo = mid + 1
            else: hi = mid - 1
        return False
