Missing Number

Easy 🔗 LeetCode

📜 Problem Statement

Given an array nums containing n distinct numbers in the range [0, n], return the only number in the range that is missing from the array.

Example: nums = [3, 0, 1] → Output: 2

Constraints: n == nums.length, 0 ≤ nums[i] ≤ n, all values are unique.

🔢 Mathematical Foundation

Sum Formula: missing = n(n+1)/2 - Σ(nums)

The expected sum of 0 to n is n(n+1)/2. The actual sum of the array is missing one number. The difference is the answer.

XOR Property: a ^ a = 0, a ^ 0 = a

XOR all numbers 0..n with all array elements. Paired numbers cancel out, leaving only the missing one.

💡 Approach Progression: Brute Force → Optimal

Approach 1
Sorting

Sort the array and look for the first index where nums[i] != i.

function missingNumber(nums):
  sort(nums)
  for i = 0 to len(nums)-1:
    if nums[i] != i:
      return i
  return len(nums)
Time: O(n log n) Space: O(1)
Pros
  • Intuitive
  • Easy to implement
Cons
  • O(n log n) due to sorting
  • Modifies the input array
Why do we need a better approach?
Sorting is overkill. We only need to find one missing element — we can use math or XOR.
Approach 2
Hash Set

Put all elements in a set, then check which number from 0 to n is not in the set.

function missingNumber(nums):
  numSet = Set(nums)
  for i = 0 to len(nums):
    if i not in numSet:
      return i
Time: O(n) Space: O(n)
Pros
  • O(n) time
  • Doesn't modify input
Cons
  • O(n) extra space for the set
Why do we need a better approach?
We need O(n) extra space. Can we do it in O(1) space?
Approach 3
XOR / Sum Formula

XOR all indices 0..n with all array elements. Duplicates cancel out (a ^ a = 0), leaving only the missing number. Alternatively, use the Gauss sum formula.

// XOR approach
function missingNumber(nums):
  xor = len(nums)
  for i = 0 to len(nums)-1:
    xor = xor ^ i ^ nums[i]
  return xor
Time: O(n) Space: O(1)
Pros
  • O(n) time, O(1) space
  • No overflow risk with XOR approach
  • Elegant use of mathematical properties
Cons
  • Less intuitive for beginners

👀 Step-by-Step Walkthrough

Trace (XOR): nums = [3, 0, 1], n = 3
Start: xor = 3 (n)
i=0: xor = 3 ^ 0 ^ 3 = 0
i=1: xor = 0 ^ 1 ^ 0 = 1
i=2: xor = 1 ^ 2 ^ 1 = 2
Result: 2 (the missing number!)
Trace (Sum): nums = [3, 0, 1], n = 3
Expected sum = 3 × 4 / 2 = 6
Actual sum = 3 + 0 + 1 = 4
Missing = 6 - 4 = 2

💻 Optimal Implementation

class Solution {
public:
    int missingNumber(vector<int>& nums) {
        int xorVal = nums.size();
        for (int i = 0; i < nums.size(); i++) {
            xorVal ^= i ^ nums[i];
        }
        return xorVal;
    }
};
class Solution:
    def missingNumber(self, nums: list[int]) -> int:
        xor = len(nums)
        for i, num in enumerate(nums):
            xor ^= i ^ num
        return xor
