Reverse Linked List

Easy 🔗 LeetCode

📜 Problem Statement

Reverse a singly linked list and return the new head.

Example: 1→2→3→4→5 → 5→4→3→2→1

🔢 Mathematical Foundation

Reversal requires redirecting each node's next pointer to its previous node. We need to store the previous, current, and next nodes to avoid losing references.

💡 Approach Progression: Brute Force → Optimal

Approach 1
Store in Array, Rebuild

Traverse the list, store all values in an array, then build a new reversed list.

function reverse(head):
  arr = []
  while head: arr.append(head.val); head = head.next
  build new list from reversed arr
Time: O(n) Space: O(n)
Pros
  • Very simple to understand
Cons
  • Uses O(n) extra space
  • Creates a new list instead of modifying in-place
Why do we need a better approach?
We don't need extra space. We can reverse the pointers in-place.
Approach 2
Iterative In-Place Reversal

Use three pointers: prev, curr, next. At each step, reverse curr's pointer, then advance all three.

function reverse(head):
  prev = null, curr = head
  while curr != null:
    next = curr.next // save next
    curr.next = prev // REVERSE pointer
    prev = curr // advance prev
    curr = next // advance curr
  return prev // new head
Time: O(n) Space: O(1)
Pros
  • O(1) extra space
  • In-place reversal
  • Foundation for many linked list problems
Cons
  • Need to be careful with pointer order

💻 Optimal Implementation

class Solution {
public:
    ListNode* reverseList(ListNode* head) {
        ListNode* prev = nullptr, *curr = head;
        while (curr) {
            ListNode* next = curr->next;
            curr->next = prev;
            prev = curr;
            curr = next;
        }
        return prev;
    }
};
class Solution:
    def reverseList(self, head):
        prev, curr = None, head
        while curr:
            nxt = curr.next
            curr.next = prev
            prev = curr
            curr = nxt
        return prev
