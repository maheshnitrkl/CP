Implement Stack using Queues

Easy 🔗 LeetCode

📜 Problem Statement

Implement a last-in-first-out (LIFO) stack using only two queues.

💡 Approach Progression: Brute Force → Optimal

Approach 1
Two Queues (Push O(1), Pop O(n))

To push, just enqueue to q1. To pop, dequeue all elements except the last one from q1 and push them to q2. The last element is the popped element. Then swap q1 and q2.

class MyStack:
  q1 = Queue(), q2 = Queue()
  function push(x): q1.enqueue(x)
  function pop():
    while q1.size() > 1: q2.enqueue(q1.dequeue())
    res = q1.dequeue()
    swap(q1, q2)
    return res
Time: Push: O(1), Pop: O(n) Space: O(n)
Pros
  • Fast Push
Cons
  • Slow Pop
  • Requires 2 Queues
Why optimize?
We can shift the O(n) cost to Push instead of Pop, or even better, eliminate the second queue entirely.
Approach 2
Two Queues (Push O(n), Pop O(1))

To push x, enqueue it to q2. Then transfer all elements from q1 to q2. Finally, swap q1 and q2. This ensures q1 is always ordered like a stack.

class MyStack:
  q1 = Queue(), q2 = Queue()
  function push(x):
    q2.enqueue(x)
    while not q1.empty(): q2.enqueue(q1.dequeue())
    swap(q1, q2)
  function pop(): return q1.dequeue()
Time: Push: O(n), Pop: O(1) Space: O(n)
Pros
  • Fast Pop
Cons
  • Slow Push
  • Still requires 2 Queues
Why optimize?
We can achieve the same logic using just one queue by rotating elements.
Approach 3
One Queue (Push O(n), Pop O(1))

When pushing x, enqueue it, then dequeue all previous elements and re-enqueue them behind x.

class MyStack:
  q = Queue()
  function push(x):
    q.enqueue(x)
    for i = 0 to q.size() - 2:
      q.enqueue(q.dequeue()) // rotate
  function pop(): return q.dequeue()
Time: Push: O(n), Pop: O(1) Space: O(n)
Pros
  • Uses only 1 queue
  • Pop is O(1)
Cons
  • Push is O(n)

👀 Step-by-Step Walkthrough

Trace: push(1), push(2), push(3)
1. push(1):
   Enqueue 1. Q = [1]. Rotations = 0.
   Stack is [1] (Top is 1).

2. push(2):
   Enqueue 2. Q = [1, 2].
   Rotate 1 time: Dequeue 1, Enqueue 1. Q = [2, 1].
   Stack is [1, 2] (Top is 2).

3. push(3):
   Enqueue 3. Q = [2, 1, 3].
   Rotate 2 times: 
     - Dequeue 2, Enqueue 2 → Q = [1, 3, 2]
     - Dequeue 1, Enqueue 1 → Q = [3, 2, 1]
   Stack is [1, 2, 3] (Top is 3).

Result: Queue perfectly mimics LIFO Stack.

💻 Optimal Implementation

class MyStack {
    queue<int> q;
public:
    void push(int x) {
        q.push(x);
        for(int i = 0; i < q.size() - 1; i++) {
            q.push(q.front()); q.pop();
        }
    }
    int pop() { int t = q.front(); q.pop(); return t; }
    int top() { return q.front(); }
    bool empty() { return q.empty(); }
};
from collections import deque
class MyStack:
    def __init__(self):
        self.q = deque()
    def push(self, x):
        self.q.append(x)
        for _ in range(len(self.q) - 1):
            self.q.append(self.q.popleft())
    def pop(self): return self.q.popleft()
    def top(self): return self.q[0]
    def empty(self): return not self.q
