Find Peak Element

Medium 🔗 LeetCode

📜 Problem Statement

A peak element is strictly greater than its neighbors. Given an array, find a peak element and return its index. The array may contain multiple peaks; return the index of any peak.

Example: [1,2,3,1] → 2 (element 3 is a peak)

Note: nums[-1] = nums[n] = -∞

🔢 Mathematical Foundation

If nums[mid] < nums[mid+1], a peak must exist to the right.

The array boundaries are -∞. If we move toward the ascending side, we must eventually find a peak (or reach the edge, which is also a peak due to -∞ boundary).

💡 Approach Progression: Brute Force → Optimal

Approach 1
Linear Scan

Check each element to see if it's greater than both neighbors.

for i = 0 to n-1:
  if (i == 0 or nums[i] > nums[i-1]) and
     (i == n-1 or nums[i] > nums[i+1]):
    return i
Time: O(n) Space: O(1)
Pros
  • Simple
  • Finds the first peak
Cons
  • Linear time
Why do we need a better approach?
We can use binary search because the peak must be on the 'ascending' side.
Approach 2
Binary Search on Answer

Compare mid with mid+1. If ascending (nums[mid] < nums[mid+1]), peak is to the right. Otherwise, peak is at mid or to the left.

function findPeak(nums):
  lo = 0, hi = n-1
  while lo < hi:
    mid = lo + (hi - lo) / 2
    if nums[mid] < nums[mid+1]:
      lo = mid + 1 // peak is right
    else:
      hi = mid // peak is at mid or left
  return lo
Time: O(log n) Space: O(1)
Pros
  • O(log n) time
  • Elegant and concise
Cons
  • Only finds one peak, not necessarily the global max

💻 Optimal Implementation

class Solution {
public:
    int findPeakElement(vector<int>& nums) {
        int lo = 0, hi = nums.size() - 1;
        while (lo < hi) {
            int mid = lo + (hi - lo) / 2;
            if (nums[mid] < nums[mid + 1]) lo = mid + 1;
            else hi = mid;
        }
        return lo;
    }
};
class Solution:
    def findPeakElement(self, nums):
        lo, hi = 0, len(nums) - 1
        while lo < hi:
            mid = (lo + hi) // 2
            if nums[mid] < nums[mid + 1]: lo = mid + 1
            else: hi = mid
        return lo
