Evaluate Reverse Polish Notation

Medium 🔗 LeetCode

📜 Problem Statement

Evaluate the value of an arithmetic expression in Reverse Polish Notation. Valid operators are +, -, *, and /.

Example: ["2","1","+","3","*"] → ((2 + 1) * 3) = 9

🔢 Mathematical Foundation

Postfix Notation: Operators follow their operands.

Using a Stack: Push operands. When an operator is seen, pop two operands, apply the operator, and push the result back.

💡 Approach Progression: Brute Force → Optimal

Approach 1
Recursion / Find and Replace

Repeatedly scan the array for the first operator, apply it to the two preceding numbers, replace the three tokens with the result.

function evalRPN(tokens):
  while len(tokens) > 1:
    i = findFirstOperator(tokens)
    val = apply(tokens[i], tokens[i-2], tokens[i-1])
    tokens = tokens[0..i-3] + [val] + tokens[i+1..]
  return tokens[0]
Time: O(n2) Space: O(n)
Pros
  • Simple concept
Cons
  • O(n^2) due to array shifting
  • Modifies array repeatedly
Why do we need a better approach?
A Stack is the perfect data structure for this, processing in O(n) time with a single pass.
Approach 2
Stack

Iterate tokens. Push numbers to stack. On operator, pop top two numbers, evaluate, and push result. Final answer is the only item left in stack.

function evalRPN(tokens):
  stack = []
  for t in tokens:
    if isNumber(t): stack.push(t)
    else:
      b = stack.pop(), a = stack.pop()
      if t == '+': stack.push(a + b)
      elif t == '-': stack.push(a - b)
      elif t == '*': stack.push(a * b)
      elif t == '/': stack.push(trunc(a / b))
  return stack.pop()
Time: O(n) Space: O(n)
Pros
  • Single pass
  • Classic use case for a Stack
  • O(n) time and space
Cons
  • Integer division truncation rules can be tricky

💻 Optimal Implementation

class Solution {
public:
    int evalRPN(vector<string>& tokens) {
        stack<long> st;
        for (string& t : tokens) {
            if (t == "+" || t == "-" || t == "*" || t == "/") {
                long b = st.top(); st.pop();
                long a = st.top(); st.pop();
                if (t == "+") st.push(a + b);
                else if (t == "-") st.push(a - b);
                else if (t == "*") st.push(a * b);
                else st.push(a / b);
            } else st.push(stol(t));
        }
        return st.top();
    }
};
class Solution:
    def evalRPN(self, tokens):
        stack = []
        for t in tokens:
            if t not in {"+", "-", "*", "/"}:
                stack.append(int(t))
            else:
                b, a = stack.pop(), stack.pop()
                if t == "+": stack.append(a + b)
                elif t == "-": stack.append(a - b)
                elif t == "*": stack.append(a * b)
                else: stack.append(int(float(a) / b))
        return stack[0]
