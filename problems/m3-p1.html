Find First and Last Position of Element in Sorted Array

Medium 🔗 LeetCode

📜 Problem Statement

Given a sorted array of integers nums and a target value, find the starting and ending position of target. If not found, return [-1, -1].

Example: nums = [5,7,7,8,8,10], target = 8 → [3, 4]

Constraint: Must run in O(log n) time.

🔢 Mathematical Foundation

Binary Search finds an element in O(log n) by halving the search space each step.

For the first occurrence, when we find target, we keep searching left (set high = mid - 1). For the last occurrence, we keep searching right (set low = mid + 1).

💡 Approach Progression: Brute Force → Optimal

Approach 1
Linear Scan

Scan left to right to find first occurrence, then continue to find last occurrence.

function searchRange(nums, target):
  first = -1, last = -1
  for i = 0 to n-1:
    if nums[i] == target:
      if first == -1: first = i
      last = i
  return [first, last]
Time: O(n) Space: O(1)
Pros
  • Very simple
  • Always works
Cons
  • Doesn't use the sorted property at all
  • O(n) is too slow for the constraint
Why do we need a better approach?
The array is sorted! We're ignoring this structure. Binary search can find elements in O(log n).
Approach 2
Two Binary Searches

Run binary search twice: once configured to find the leftmost occurrence (when found, keep going left), once for the rightmost (when found, keep going right).

function searchRange(nums, target):
  return [findBound(nums, target, true),
          findBound(nums, target, false)]

function findBound(nums, target, isFirst):
  lo = 0, hi = n-1, result = -1
  while lo ≤ hi:
    mid = lo + (hi - lo) / 2
    if nums[mid] == target:
      result = mid
      if isFirst: hi = mid - 1 // keep searching left
      else: lo = mid + 1 // keep searching right
    elif nums[mid] < target: lo = mid + 1
    else: hi = mid - 1
  return result
Time: O(log n) Space: O(1)
Pros
  • O(log n) time
  • Meets the constraint
  • Clean separation of concerns
Cons
  • Slightly more complex than naive binary search

💻 Optimal Implementation

class Solution {
public:
    vector<int> searchRange(vector<int>& nums, int target) {
        return {findBound(nums, target, true), findBound(nums, target, false)};
    }
    int findBound(vector<int>& nums, int target, bool isFirst) {
        int lo = 0, hi = nums.size()-1, result = -1;
        while (lo <= hi) {
            int mid = lo + (hi-lo)/2;
            if (nums[mid] == target) {
                result = mid;
                if (isFirst) hi = mid - 1;
                else lo = mid + 1;
            } else if (nums[mid] < target) lo = mid + 1;
            else hi = mid - 1;
        }
        return result;
    }
};
class Solution:
    def searchRange(self, nums, target):
        def find(is_first):
            lo, hi, result = 0, len(nums)-1, -1
            while lo <= hi:
                mid = (lo + hi) // 2
                if nums[mid] == target:
                    result = mid
                    if is_first: hi = mid - 1
                    else: lo = mid + 1
                elif nums[mid] < target: lo = mid + 1
                else: hi = mid - 1
            return result
        return [find(True), find(False)]
