Merge Two Sorted Lists

Easy 🔗 LeetCode

📜 Problem Statement

Merge two sorted linked lists into one sorted list by splicing together the nodes.

Example: 1→2→4 + 1→3→4 → 1→1→2→3→4→4

💡 Approach Progression: Brute Force → Optimal

Approach 1
Collect, Sort, Rebuild

Extract all values from both lists, sort them, create a new list.

function mergeTwoLists(l1, l2):
  arr = collectAll(l1) + collectAll(l2)
  sort(arr)
  return buildList(arr)
Time: O(n log n) Space: O(n)
Pros
  • Simple
Cons
  • Ignores that both lists are already sorted
  • O(n log n) sorting is wasteful
Why do we need a better approach?
Both lists are already sorted! We can merge in O(n) by comparing heads.
Approach 2
Iterative Merge with Dummy Node

Use a dummy head node. Compare current nodes of both lists, attach the smaller one to the merged list. Advance that list's pointer.

function mergeTwoLists(l1, l2):
  dummy = ListNode(0)
  tail = dummy
  while l1 AND l2:
    if l1.val ≤ l2.val:
      tail.next = l1; l1 = l1.next
    else:
      tail.next = l2; l2 = l2.next
    tail = tail.next
  tail.next = l1 or l2 // attach remaining
  return dummy.next
Time: O(n + m) Space: O(1)
Pros
  • O(n+m) time
  • In-place splicing, no new nodes
  • Classic two-pointer merge
Cons
  • Need to handle remaining nodes

👀 Step-by-Step Walkthrough

Trace: l1 = [1, 2, 4], l2 = [1, 3, 4]
Initialize: dummy = ListNode(0), tail = dummy

Step 1: l1(1) ≤ l2(1)
  tail.next = l1 (1)
  tail moves to 1 (from l1), l1 moves to 2

Step 2: l1(2) > l2(1)
  tail.next = l2 (1)
  tail moves to 1 (from l2), l2 moves to 3

Step 3: l1(2) ≤ l2(3)
  tail.next = l1 (2)
  tail moves to 2, l1 moves to 4

Step 4: l1(4) > l2(3)
  tail.next = l2 (3)
  tail moves to 3, l2 moves to 4

Step 5: l1(4) ≤ l2(4)
  tail.next = l1 (4)
  tail moves to 4 (from l1), l1 moves to null

Loop ends (l1 is null).
Attach remaining: tail.next = l2 (which is 4)

Result: dummy.next → 1 → 1 → 2 → 3 → 4 → 4

💻 Optimal Implementation

class Solution {
public:
    ListNode* mergeTwoLists(ListNode* l1, ListNode* l2) {
        ListNode dummy(0), *tail = &dummy;
        while (l1 && l2) {
            if (l1->val <= l2->val) { tail->next = l1; l1 = l1->next; }
            else { tail->next = l2; l2 = l2->next; }
            tail = tail->next;
        }
        tail->next = l1 ? l1 : l2;
        return dummy.next;
    }
};
