Sort List

Medium 🔗 LeetCode

📜 Problem Statement

Sort a linked list in O(n log n) time and O(1) memory.

💡 Approach Progression: Brute Force → Optimal

Approach 1
Extract, Sort, Rebuild

Extract values to array, sort, rebuild list.

function sortList(head):
  arr = extractValues(head)
  sort(arr)
  return buildList(arr)
Time: O(n log n) Space: O(n)
Pros
  • Simple
Cons
  • O(n) extra space
Why do we need a better approach?
We can sort in-place using Merge Sort on the linked list.
Approach 2
Merge Sort (Recursive)

Find middle, split into two halves, recursively sort each half, merge the sorted halves.

function sortList(head):
  if head == null OR head.next == null: return head
  mid = findMiddle(head) // split
  right = mid.next; mid.next = null
  left = sortList(head) // sort left half
  right = sortList(right) // sort right half
  return merge(left, right) // merge sorted halves
Time: O(n log n) Space: O(log n) stack
Pros
  • O(n log n) guaranteed
  • Stable sort
  • Natural for linked lists (no random access needed)
Cons
  • O(log n) stack space for recursion

💻 Optimal Implementation

class Solution {
public:
    ListNode* sortList(ListNode* head) {
        if (!head || !head->next) return head;
        ListNode *slow = head, *fast = head->next;
        while (fast && fast->next) { slow = slow->next; fast = fast->next->next; }
        ListNode *right = slow->next; slow->next = nullptr;
        return merge(sortList(head), sortList(right));
    }
    ListNode* merge(ListNode* l1, ListNode* l2) {
        ListNode dummy(0), *tail = &dummy;
        while (l1 && l2) {
            if (l1->val <= l2->val) { tail->next = l1; l1 = l1->next; }
            else { tail->next = l2; l2 = l2->next; }
            tail = tail->next;
        }
        tail->next = l1 ? l1 : l2;
        return dummy.next;
    }
};
class Solution:
    def sortList(self, head):
        if not head or not head.next: return head
        slow, fast = head, head.next
        while fast and fast.next: slow = slow.next; fast = fast.next.next
        right = slow.next; slow.next = None
        left = self.sortList(head)
        right = self.sortList(right)
        dummy = tail = ListNode(0)
        while left and right:
            if left.val <= right.val: tail.next = left; left = left.next
            else: tail.next = right; right = right.next
            tail = tail.next
        tail.next = left or right
        return dummy.next
