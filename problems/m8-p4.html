Largest Rectangle in Histogram

Hard 🔗 LeetCode

📜 Problem Statement

Given an array of integers representing the histogram's bar height where the width of each bar is 1, return the area of the largest rectangle in the histogram.

🔢 Mathematical Foundation

Area = height[i] × (right_smaller_idx - left_smaller_idx - 1)

For each bar, if it is the minimum height of a rectangle, the rectangle extends left and right until it hits a bar smaller than it.

💡 Approach Progression: Brute Force → Optimal

Approach 1
Expand from Each Bar

For every bar, expand left and right to find the boundaries where bars are ≥ current height.

for i = 0 to n-1:   expand left and right   maxArea = max(maxArea, area)
Time: O(n2) Space: O(1)
Pros
  • Intuitive
Cons
  • Too slow
Why do we need a better approach?
A monotonic stack can find the left and right smaller elements for all bars in a single pass O(n).
Approach 2
Monotonic Increasing Stack

Maintain a stack of indices with increasing heights. When a smaller bar is encountered, it acts as the 'right boundary' for the bars in the stack. The 'left boundary' is the element below it in the stack.

function largestRectangleArea(heights):
  stack = []; maxArea = 0
  heights.append(0) // flush remaining bars
  for i, h in enumerate(heights):
    while stack and h < heights[stack.top()]:
      height = heights[stack.pop()]
      width = i if stack.empty() else i - stack.top() - 1
      maxArea = max(maxArea, height * width)
    stack.push(i)
  return maxArea
Time: O(n) Space: O(n)
Pros
  • O(n) time
  • Masterclass in Monotonic Stack
Cons
  • Very tricky to handle width calculations correctly

💻 Optimal Implementation

class Solution {
public:
    int largestRectangleArea(vector<int>& heights) {
        heights.push_back(0);
        stack<int> st;
        int maxArea = 0;
        for (int i = 0; i < heights.size(); i++) {
            while (!st.empty() && heights[i] < heights[st.top()]) {
                int h = heights[st.top()]; st.pop();
                int w = st.empty() ? i : i - st.top() - 1;
                maxArea = max(maxArea, h * w);
            }
            st.push(i);
        }
        return maxArea;
    }
};
class Solution:
    def largestRectangleArea(self, heights):
        heights.append(0)
        stack = []
        maxArea = 0
        for i, h in enumerate(heights):
            while stack and h < heights[stack[-1]]:
                height = heights[stack.pop()]
                width = i if not stack else i - stack[-1] - 1
                maxArea = max(maxArea, height * width)
            stack.append(i)
        return maxArea
