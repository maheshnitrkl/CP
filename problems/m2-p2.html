Bit Difference

Easy 🔗 GFG

📜 Problem Statement

Given two integers a and b, count the number of bits that need to be flipped to convert a to b. This is the Hamming Distance between two numbers.

Example: a = 10 (1010), b = 20 (10100) → Output: 4

🔢 Mathematical Foundation

Hamming Distance = popcount(a XOR b)

XOR produces 1 only where bits differ. Counting those 1s gives the number of flips needed.

Example: 10 XOR 20 = 1010 XOR 10100 = 11110 → 4 bits set → 4 flips needed.

💡 Approach Progression: Brute Force → Optimal

Approach 1
Compare Bit by Bit

Extract each bit of both numbers using right shifts, compare them one by one.

function bitDiff(a, b):
  count = 0
  for i = 0 to 31:
    bitA = (a >> i) & 1
    bitB = (b >> i) & 1
    if bitA != bitB: count += 1
  return count
Time: O(32) Space: O(1)
Pros
  • Easy to understand
  • No bitwise tricks needed
Cons
  • Always 32 iterations
  • Two operations per iteration
Why do we need a better approach?
We're doing two extractions per bit. XOR can tell us all differing positions in a single operation.
Approach 2
XOR + Brian Kernighan's

XOR gives us a number where only the differing bits are set. Then count those set bits using Brian Kernighan's algorithm.

function bitDiff(a, b):
  xor = a ^ b // 1s where bits differ
  count = 0
  while xor != 0:
    xor = xor & (xor - 1)
    count += 1
  return count
Time: O(k) where k = differing bits Space: O(1)
Pros
  • Only iterates over differing bits
  • Elegant composition of two techniques
Cons
  • Requires understanding of XOR and Kernighan's

👀 Step-by-Step Walkthrough

Trace: a = 10 (01010), b = 20 (10100)
Step 1: xor = 01010 ^ 10100 = 11110 (4 bits set)
Step 2: xor & (xor-1) = 11110 & 11101 = 11100, count = 1
Step 3: 11100 & 11011 = 11000, count = 2
Step 4: 11000 & 10111 = 10000, count = 3
Step 5: 10000 & 01111 = 00000, count = 4

💻 Optimal Implementation

class Solution {
public:
    int countBitsFlip(int a, int b) {
        int xorVal = a ^ b, count = 0;
        while (xorVal) {
            xorVal &= (xorVal - 1);
            count++;
        }
        return count;
    }
};

    def countBitsFlip(self, a, b):
