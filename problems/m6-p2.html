Middle of the Linked List

Easy 🔗 LeetCode

📜 Problem Statement

Return the middle node of a linked list. If two middle nodes, return the second one.

Example: 1→2→3→4→5 → node 3

🔢 Mathematical Foundation

Slow/Fast Pointer Technique (Tortoise & Hare)

Slow moves 1 step, fast moves 2 steps. When fast reaches the end, slow is at the middle. This works because slow covers half the distance of fast.

💡 Approach Progression: Brute Force → Optimal

Approach 1
Count, Then Traverse

First pass: count nodes (n). Second pass: traverse to node n/2.

function middleNode(head):
  count = countNodes(head) // pass 1
  mid = count / 2
  for i = 0 to mid-1: head = head.next // pass 2
  return head
Time: O(n) Space: O(1)
Pros
  • Simple logic
  • Two passes
Cons
  • Two passes through the list
Why do we need a better approach?
Slow/Fast pointers find the middle in a single pass.
Approach 2
Slow & Fast Pointers

Slow advances 1 step, fast advances 2 steps. When fast reaches the end, slow is at the middle.

function middleNode(head):
  slow = fast = head
  while fast != null AND fast.next != null:
    slow = slow.next
    fast = fast.next.next
  return slow
Time: O(n) Space: O(1)
Pros
  • Single pass
  • No need to know length
  • Elegant
Cons
  • Slightly less intuitive

👀 Step-by-Step Walkthrough

Trace: 1 → 2 → 3 → 4 → 5
Initialize: slow = 1, fast = 1

Step 1:
  slow moves to 2
  fast moves to 3

Step 2:
  slow moves to 3
  fast moves to 5

Step 3:
  fast.next is null. Loop ends.

Result: slow is at node 3 (the middle).

💻 Optimal Implementation

class Solution {
public:
    ListNode* middleNode(ListNode* head) {
        ListNode* slow = head, *fast = head;
        while (fast && fast->next) {
            slow = slow->next;
            fast = fast->next->next;
        }
        return slow;
    }
};
class Solution:
    def middleNode(self, head):
        slow = fast = head
        while fast and fast.next:
            slow = slow.next
            fast = fast.next.next
        return slow
