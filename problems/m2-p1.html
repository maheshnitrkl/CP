Number of 1 Bits

Easy 🔗 LeetCode

📜 Problem Statement

Given a positive integer n, return the number of set bits (1s) in its binary representation. This is also known as the Hamming Weight.

Example: n = 11 (binary: 1011) → Output: 3

Constraints: 1 ≤ n ≤ 231 - 1

🔢 Mathematical Foundation

n & (n - 1)

This operation clears the lowest set bit of n.

Example: n = 12 (1100), n-1 = 11 (1011), n & (n-1) = 8 (1000)

Each application removes exactly one '1' bit. So we count how many times we can do this until n becomes 0.

💡 Approach Progression: Brute Force → Optimal

Approach 1
Bit-by-Bit Check

The most natural approach: examine each of the 32 bits one by one. Right-shift n and check if the last bit is 1 using n & 1.

function hammingWeight(n):
  count = 0
  for i = 0 to 31:
    if n & 1 == 1: // check last bit
      count += 1
    n = n >> 1 // shift right
  return count
Time: O(32) = O(1) Space: O(1)
Pros
  • Dead simple to understand
  • Always checks exactly 32 iterations
  • No tricky bit operations
Cons
  • Always runs 32 iterations even if n=1 (only 1 bit set)
  • Doesn't leverage the structure of the number
Why do we need a better approach?
For n = 1 (binary: 0...001), we waste 31 iterations checking zeros. Can we do better by only visiting the set bits?
Approach 2
Brian Kernighan's Algorithm

The key insight: n & (n-1) clears the lowest set bit. We repeatedly apply this until n becomes 0. The number of iterations equals the number of set bits — we never waste time on zero bits!

function hammingWeight(n):
  count = 0
  while n != 0:
    n = n & (n - 1) // clear lowest set bit
    count += 1
  return count
Time: O(k) where k = number of set bits Space: O(1)
Pros
  • Only iterates k times (number of 1-bits)
  • Elegant and efficient
  • Leverages bitwise properties
Cons
  • Slightly less intuitive for beginners

Iteration 2: n = 1010, n-1 = 1001, n & (n-1) = 1000, count = 2
Iteration 3: n = 1000, n-1 = 0111, n & (n-1) = 0000, count = 3
Result: n = 0, return 3. Only 3 iterations for 3 set bits!

💻 Optimal Implementation

class Solution {
public:
    int hammingWeight(int n) {
        int count = 0;
        while (n) {
            n &= (n - 1);  // clear lowest set bit
            count++;
        }
        return count;
    }
};
