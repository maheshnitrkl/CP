Daily Temperatures

Medium 🔗 LeetCode

📜 Problem Statement

Given an array of temperatures, return an array such that answer[i] is the number of days you have to wait after the ith day to get a warmer temperature. If none exists, put 0.

Example: [73,74,75,71,69,72,76,73] → [1,1,4,2,1,1,0,0]

🔢 Mathematical Foundation

Monotonic Stack

Maintain a stack of indices corresponding to temperatures in decreasing order. When a warmer day arrives, it resolves the wait time for all colder days currently in the stack.

💡 Approach Progression: Brute Force → Optimal

Approach 1
Nested Loops

For each day, scan forward until a warmer day is found.

function dailyTemperatures(T):
  ans = array of 0s
  for i = 0 to n-1:
    for j = i+1 to n-1:
      if T[j] > T[i]:
        ans[i] = j - i; break
  return ans
Time: O(n2) Space: O(1)
Pros
  • Simple logic
Cons
  • O(n^2) is too slow for large inputs
Why do we need a better approach?
We re-scan the same temperatures repeatedly. A Monotonic Stack can process each element exactly twice (push and pop) for O(n).
Approach 2
Monotonic Decreasing Stack

Iterate through temperatures. While current temp is > temp at stack.top(), pop the stack. The wait time for the popped day is curr_idx - popped_idx. Then push current index.

function dailyTemperatures(T):
  ans = array of 0s, stack = []
  for i, t in enumerate(T):
    while stack and t > T[stack.top()]:
      prev = stack.pop()
      ans[prev] = i - prev
    stack.push(i)
  return ans
Time: O(n) Space: O(n)
Pros
  • O(n) time — each index pushed/popped once
  • Classic Monotonic Stack pattern
Cons
  • Stack stores indices, not values (can be confusing)

💻 Optimal Implementation

class Solution {
public:
    vector<int> dailyTemperatures(vector<int>& T) {
        vector<int> ans(T.size(), 0);
        stack<int> st; // stores indices
        for (int i = 0; i < T.size(); i++) {
            while (!st.empty() && T[i] > T[st.top()]) {
                int prev = st.top(); st.pop();
                ans[prev] = i - prev;
            }
            st.push(i);
        }
        return ans;
    }
};
class Solution:
    def dailyTemperatures(self, T):
        ans = [0] * len(T)
        stack = []
        for i, t in enumerate(T):
            while stack and t > T[stack[-1]]:
                prev = stack.pop()
                ans[prev] = i - prev
            stack.append(i)
        return ans
