Make them Co-prime

Easy 🔗 GFG

📝 Problem Statement

Find the optimal solution for this problem.

🔢 Mathematical Foundation

GCD Property: GCD(a·g, b·g) = g × GCD(a, b)

If all elements in an array share a common factor g, dividing every element by g removes that common factor from all pairs simultaneously.

By finding the global GCD of the entire array and dividing all elements by it, the new global GCD becomes 1, making the elements co-prime.

💡 Approach Progression: Brute Force → Optimal

Approach 1
Check Every Possible Divisor

Find the minimum element in the array. Try dividing all elements by every integer from 2 up to that minimum element to see if there is a common divisor.

Time: O(N × min(arr)) Space: O(1)
Pros
  • Conceptual simplicity
Cons
  • Too slow if the minimum element is large
Why do we need a better approach?
Finding the Greatest Common Divisor using the Euclidean algorithm is significantly faster than checking every number manually.
Approach 2
Global GCD Division

Iterate through the array to find the global Greatest Common Divisor (GCD) of all elements. Divide every element by this global GCD.

function gcd(a, b):
  return b == 0 ? a : gcd(b, a % b)

global_g = arr[0]
for x in arr:
  global_g = gcd(global_g, x)
for i = 0 to N-1:
  arr[i] = arr[i] / global_g
Time: O(N × log(min(a,b))) Space: O(1)
Pros
  • Optimal time complexity
Cons
  • None

💻 Optimal Implementation

class Solution {
public:
    int gcd(int a, int b) {
        return b == 0 ? a : gcd(b, a % b);
    }
    
    vector<int> makeCoPrime(vector<int>& arr) {
        if (arr.empty()) return arr;
        
        int g = arr[0];
        for (int i = 1; i < arr.size(); i++) {
            g = gcd(g, arr[i]);
        }
        
        for (int i = 0; i < arr.size(); i++) {
            arr[i] /= g;
        }
        return arr;
    }
};
import math
class Solution:
    def makeCoPrime(self, arr):
        if not arr: return arr
        
        g = arr[0]
        for x in arr[1:]:
            g = math.gcd(g, x)
            
        return [x // g for x in arr]
