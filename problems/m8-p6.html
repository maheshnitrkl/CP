Implement Queue using Stacks

Easy 🔗 LeetCode

📜 Problem Statement

Implement a first-in-first-out (FIFO) queue using only two stacks.

💡 Approach Progression: Brute Force → Optimal

Approach 1
Two Stacks (Push O(n), Pop O(1))

To enqueue an element, move all elements from Stack1 to Stack2, push the new element to Stack1, and then move all elements back to Stack1. This keeps the oldest element at the top.

class MyQueue:
  s1 = [], s2 = []
  function push(x):
    while not s1.empty(): s2.push(s1.pop())
    s1.push(x)
    while not s2.empty(): s1.push(s2.pop())
  function pop(): return s1.pop()
Time: Push: O(n), Pop: O(1) Space: O(n)
Pros
  • Fast Pop O(1)
Cons
  • Push is O(n)
  • Constant moving back and forth
Why optimize?
We can avoid moving elements back and forth. Elements poured into Stack2 are already in Queue order! We only need to pour when Stack2 is empty.
Approach 2
Two Stacks (Push O(1), Pop Amortized O(1))

Stack1 for enqueuing. When dequeuing, if Stack2 is empty, pop all elements from Stack1 and push to Stack2 (reverses order). Pop from Stack2.

class MyQueue:
  inStack = [], outStack = []
  function push(x): inStack.push(x)
  function pop():
    shift()
    return outStack.pop()
  function shift():
    if outStack.empty():
      while not inStack.empty():
        outStack.push(inStack.pop())
Time: Push: O(1), Pop: Amortized O(1) Space: O(n)
Pros
  • Amortized O(1) for pop
  • Push is strictly O(1)
Cons
  • Worst case pop is O(n)

💻 Optimal Implementation

class MyQueue {
    stack<int> in, out;
    void shift() {
        if (out.empty()) {
            while (!in.empty()) { out.push(in.top()); in.pop(); }
        }
    }
public:
    void push(int x) { in.push(x); }
    int pop() { shift(); int t = out.top(); out.pop(); return t; }
    int peek() { shift(); return out.top(); }
    bool empty() { return in.empty() && out.empty(); }
};
class MyQueue:
    def __init__(self):
        self.in_st, self.out_st = [], []
    def push(self, x): self.in_st.append(x)
    def shift(self):
        if not self.out_st:
            while self.in_st: self.out_st.append(self.in_st.pop())
    def pop(self):
        self.shift(); return self.out_st.pop()
    def peek(self):
        self.shift(); return self.out_st[-1]
    def empty(self):
        return not self.in_st and not self.out_st
