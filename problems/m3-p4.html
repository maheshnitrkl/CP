Single Element in a Sorted Array

Medium 🔗 LeetCode

📜 Problem Statement

In a sorted array where every element appears exactly twice except one, find that single element.

Example: [1,1,2,3,3,4,4,8,8] → 2

Constraint: Must run in O(log n) time and O(1) space.

🔢 Mathematical Foundation

Parity Property: Before the single element, pairs start at even indices. After it, pairs start at odd indices.

If mid is even and nums[mid] == nums[mid+1], the single element is to the right. This parity shift lets us binary search.

💡 Approach Progression: Brute Force → Optimal

Approach 1
XOR All Elements

XOR all elements. Paired elements cancel out, leaving the single one.

function singleNonDuplicate(nums):
  result = 0
  for x in nums: result ^= x
  return result
Time: O(n) Space: O(1)
Pros
  • Very simple
  • Works for unsorted arrays too
Cons
  • O(n) time — doesn't use sorted property
Why do we need a better approach?
The array is sorted! We can exploit the pair-index parity to use binary search for O(log n).
Approach 2
Binary Search with Parity Check

Ensure mid is even. If nums[mid] == nums[mid+1], pairs are intact → single is to the right. Otherwise, single is at mid or to the left.

function singleNonDuplicate(nums):
  lo = 0, hi = n-1
  while lo < hi:
    mid = lo + (hi - lo) / 2
    if mid is odd: mid -= 1 // ensure even
    if nums[mid] == nums[mid+1]:
      lo = mid + 2 // pair intact, single is right
    else:
      hi = mid // pair broken, single is here or left
  return nums[lo]
Time: O(log n) Space: O(1)
Pros
  • O(log n) time
  • Clever use of index parity
Cons
  • Harder to derive the invariant

👀 Step-by-Step Walkthrough

Trace: [1,1,2,3,3,4,4,8,8]
lo=0, hi=8: mid=4 (even). nums[4]=3, nums[5]=4. 3 ≠ 4 → pair broken, hi=4
lo=0, hi=4: mid=2 (even). nums[2]=2, nums[3]=3. 2 ≠ 3 → pair broken, hi=2
lo=0, hi=2: mid=1 (odd → adjust to 0). nums[0]=1, nums[1]=1. 1 == 1 → pair intact, lo=2
lo=2, hi=2: lo == hi → return nums[2] = 2

💻 Optimal Implementation

class Solution {
public:
    int singleNonDuplicate(vector<int>& nums) {
        int lo = 0, hi = nums.size() - 1;
        while (lo < hi) {
            int mid = lo + (hi - lo) / 2;
            if (mid % 2 == 1) mid--;
            if (nums[mid] == nums[mid + 1]) lo = mid + 2;
            else hi = mid;
        }
        return nums[lo];
    }
};
class Solution:
    def singleNonDuplicate(self, nums):
        lo, hi = 0, len(nums) - 1
        while lo < hi:
            mid = (lo + hi) // 2
            if mid % 2 == 1: mid -= 1
            if nums[mid] == nums[mid + 1]: lo = mid + 2
            else: hi = mid
        return nums[lo]
