Isomorphic Strings

Easy 🔗 LeetCode

📜 Problem Statement

Determine if string s can be replaced to get string t. No two characters may map to the same character.

💡 Approach Progression: Brute Force → Optimal

Approach 1
String Replacement Simulation

Iterate through the string. For every character `s[i]`, replace all its occurrences in a temporary string with `t[i]`. At the end, check if the replaced string equals `t`.

function isIsomorphic(s, t):
  if len(s) != len(t): return false
  temp = list(s)
  for i = 0 to len(s)-1:
    c1 = s[i], c2 = t[i]
    for j = i to len(s)-1:
      if temp[j] == c1: temp[j] = c2
  return "".join(temp) == t
Time: O(n2) Space: O(n)
Pros
  • Conceptually simple simulation
Cons
  • O(n²) time limit exceeded
  • Creates many temporary strings/arrays
Why optimize?
We don't need to actually replace the string. We just need to ensure that every character `c1` in `s` maps strictly to `c2` in `t`, and vice versa. We can track this mapping in O(1) time per character using Hash Maps.
Approach 2
Two Hash Maps

Maintain mapping from s to t, and t to s to ensure a 1-to-1 bijection. If any mapping conflicts, return false.

function isIsomorphic(s, t):
  mapS, mapT = {}, {}
  for i = 0 to n-1:
    c1, c2 = s[i], t[i]
    if mapS[c1] != c2 OR mapT[c2] != c1: return false
  return true
Time: O(n) Space: O(1) (size of alphabet)
Pros
  • Ensures 1-to-1 mapping bidirectionally
Cons
  • Need to manage 2 separate Hash Maps
Why optimize?
We can do this with just one combined "last seen index" tracking logic.
Approach 3
Single Map + Index Signature

Two strings are isomorphic if the pattern of character appearances is exactly the same. We can convert both strings into a normalized "index signature" (e.g., "paper" → [0, 1, 0, 3, 4] and "title" → [0, 1, 0, 3, 4]). If the signatures match at every step, they are isomorphic.

function isIsomorphic(s, t):
  lastSeenS = {}, lastSeenT = {}
  for i = 0 to n-1:
    if lastSeenS[s[i]] != lastSeenT[t[i]]: return false
    lastSeenS[s[i]] = i + 1
    lastSeenT[t[i]] = i + 1
  return true
Time: O(n) Space: O(1) (size of alphabet)
Pros
  • Very clean logic
  • Avoids cross-mapping errors
Cons
  • Slightly trickier to reason about initially

💻 Optimal Implementation

class Solution {
public:
    bool isIsomorphic(string s, string t) {
        vector<int> mapS(256, -1), mapT(256, -1);
        for (int i = 0; i < s.size(); i++) {
            if (mapS[s[i]] != mapT[t[i]]) return false;
            mapS[s[i]] = mapT[t[i]] = i; // Store last seen index
        }
        return true;
    }
};
class Solution:
    def isIsomorphic(self, s, t):
        return [s.find(i) for i in s] == [t.find(j) for j in t]
