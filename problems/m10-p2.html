Decode Ways

Medium 🔗 LeetCode

📜 Problem Statement

A message containing letters from A-Z can be encoded into numbers using the mapping A -> 1, B -> 2, ..., Z -> 26. Given a string s containing only digits, return the number of ways to decode it.

📚 Mathematical Foundation

Let dp[i] be the number of ways to decode the prefix of string s of length i.

When considering the i-th character, it can be decoded in two ways:

  • Single digit: If s[i-1] != '0', it forms a valid letter. We add dp[i-1].
  • Two digits: If s[i-2...i-1] forms a number between 10 and 26, it forms a valid letter. We add dp[i-2].
dp[i] = (s[i-1] != '0' ? dp[i-1] : 0) + (10 ≤ s[i-2..i-1] ≤ 26 ? dp[i-2] : 0)

💡 Approach Progression: Brute Force → Optimal

Approach 1
Brute Force Recursion

At every index, branch out to decode 1 digit and 2 digits (if valid). Add the results. Overlaps massively.

function decode(i, s):
  if i == len(s): return 1
  if s[i] == '0': return 0
  ways = decode(i+1, s)
  if i+1 < len(s) and s[i:i+2] <= "26":
    ways += decode(i+2, s)
  return ways
Time: O(2n) Space: O(n)
Pros
  • Easy to reason about
Cons
  • Time Limit Exceeded
Why optimize?
We evaluate the same suffixes repeatedly. We can memoize the recursive function.
Approach 2
Top-Down Memoization

Add an array or hash map to cache the number of ways to decode starting from index i.

function decode(i, s, memo):
  if i in memo: return memo[i]
  if i == len(s): return 1
  if s[i] == '0': return 0
  ans = decode(i+1, s, memo)
  if i+1 < len(s) and s[i:i+2] <= "26":
    ans += decode(i+2, s, memo)
  memo[i] = ans
  return ans
Time: O(n) Space: O(n)
Pros
  • Fast O(n) time
Cons
  • Call stack overhead
Why optimize?
Recursion limit could be reached. A bottom-up Tabulation approach is safer and avoids stack overhead.
Approach 3
Bottom-Up Tabulation (1D DP)

Iterate from 1 to N, building the dp array where dp[i] stores the ways to decode the first i characters.

function numDecodings(s):
  dp = array of size N+1 (all 0s)
  dp[0] = 1
  dp[1] = s[0] != '0' ? 1 : 0
  for i = 2 to N:
    if s[i-1] != '0': dp[i] += dp[i-1]
    twoDigit = int(s[i-2:i])
    if 10 <= twoDigit <= 26: dp[i] += dp[i-2]
  return dp[N]
Time: O(n) Space: O(n)
Pros
  • Iterative, no stack overflow
Cons
  • Still O(n) space
Why optimize?
Notice that dp[i] only depends on dp[i-1] and dp[i-2]. We can optimize the space to O(1) by only keeping the last two values!
Approach 4
Space-Optimized Bottom-Up

Instead of a full array, just use two variables dp1 (representing dp[i-1]) and dp2 (representing dp[i-2]).

function numDecodings(s):
  if s[0] == '0': return 0
  dp1 = 1, dp2 = 1
  for i = 1 to len(s)-1:
    dp = 0
    if s[i] != '0': dp += dp1
    if 10 <= int(s[i-1:i+1]) <= 26: dp += dp2
    dp2 = dp1, dp1 = dp
  return dp1
Time: O(n) Space: O(1)
Pros
  • Optimal space and time
Cons
  • Slightly trickier to read

💻 Optimal Implementation

class Solution {
public:
    int numDecodings(string s) {
        if (s.empty() || s[0] == '0') return 0;
        
        int dp1 = 1, dp2 = 1;
        
        for (int i = 1; i < s.length(); i++) {
            int current_dp = 0;
            
            // Single digit
            if (s[i] != '0') {
                current_dp += dp1;
            }
            
            // Double digit
            int twoDigit = stoi(s.substr(i - 1, 2));
            if (twoDigit >= 10 && twoDigit <= 26) {
                current_dp += dp2;
            }
            
            dp2 = dp1;
            dp1 = current_dp;
        }
        
        return dp1;
    }
};
class Solution:
    def numDecodings(self, s: str) -> int:
        if not s or s[0] == '0':
            return 0
            
        dp1 = 1
        dp2 = 1
        
        for i in range(1, len(s)):
            current_dp = 0
            
            # Single digit decode
            if s[i] != '0':
                current_dp += dp1
                
            # Double digit decode
            if 10 <= int(s[i-1:i+1]) <= 26:
                current_dp += dp2
                
            dp2 = dp1
            dp1 = current_dp
            
        return dp1
