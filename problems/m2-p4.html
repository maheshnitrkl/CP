Find Position of Set Bit

Easy 🔗 GFG

📜 Problem Statement

Given a number n that has only one set bit in its binary representation, find the position of that set bit (1-indexed from the right).

Example: n = 16 (10000) → Position: 5. n = 2 (10) → Position: 2.

If n is not a power of 2, return -1.

🔢 Mathematical Foundation

position = log2(n) + 1

Since n has exactly one set bit, it's a power of 2. The position is simply log2(n) + 1.

Alternatively, keep right-shifting n until it becomes 0, counting shifts.

💡 Approach Progression: Brute Force → Optimal

Approach 1
Scan All 32 Bits

Check each bit position from 0 to 31 to find which bit is set.

function findPosition(n):
  if !isPowerOfTwo(n): return -1
  for i = 0 to 31:
    if (n >> i) & 1 == 1:
      return i + 1
  return -1
Time: O(32) Space: O(1)
Pros
  • Simple logic
  • No math functions needed
Cons
  • Scans all 32 bits even if answer is at position 1
Why do we need a better approach?
We're linearly scanning when we could compute the answer directly with logarithm.
Approach 2
Log Base 2

Since n is a power of 2, we can directly compute the position using log2(n). In C++ we can also use __builtin_ctz(n) (count trailing zeros) which is a single CPU instruction.

function findPosition(n):
  if n ≤ 0 OR (n & (n-1)) != 0:
    return -1 // not a power of 2
  return log2(n) + 1
Time: O(1) Space: O(1)
Pros
  • Constant time
  • Uses built-in CPU instructions
Cons
  • log2 may have floating point precision issues

💻 Optimal Implementation

class Solution {
public:
    int findPosition(int n) {
        if (n <= 0 || (n & (n - 1)) != 0) return -1;
        return __builtin_ctz(n) + 1; // count trailing zeros
    }
};
class Solution:
    def findPosition(self, n):
        if n <= 0 or (n & (n - 1)) != 0:
            return -1
        return n.bit_length()  # position of highest set bit
