Reorder List

Medium 🔗 LeetCode

📜 Problem Statement

Reorder list: L0 → Ln → L1 → Ln-1 → L2 → ...

Example: 1→2→3→4 → 1→4→2→3

💡 Approach Progression: Brute Force → Optimal

Approach 1
Array + Interleave

Store nodes in array. Use two pointers from both ends to build new order.

function reorderList(head):
  nodes = toArray(head)
  i = 0, j = n-1
  while i < j:
    nodes[i].next = nodes[j]; i++
    nodes[j].next = nodes[i]; j--
Time: O(n) Space: O(n)
Pros
  • Simple with array
Cons
  • O(n) extra space
Why do we need a better approach?
We can do this in O(1) space using find-middle + reverse + merge.
Approach 2
Find Middle + Reverse + Merge

1) Find middle 2) Reverse second half 3) Merge/interleave both halves.

function reorderList(head):
  mid = findMiddle(head)
  second = reverse(mid.next); mid.next = null
  // interleave first and second halves
  while second:
    tmp1 = first.next; tmp2 = second.next
    first.next = second; second.next = tmp1
    first = tmp1; second = tmp2
Time: O(n) Space: O(1)
Pros
  • O(1) space
  • Combines 3 fundamental LL operations
  • Beautiful composition
Cons
  • More complex implementation

💻 Optimal Implementation

class Solution {
public:
    void reorderList(ListNode* head) {
        if (!head || !head->next) return;
        ListNode *slow = head, *fast = head;
        while (fast->next && fast->next->next) { slow = slow->next; fast = fast->next->next; }
        ListNode *prev = nullptr, *curr = slow->next; slow->next = nullptr;
        while (curr) { ListNode *nxt = curr->next; curr->next = prev; prev = curr; curr = nxt; }
        ListNode *first = head, *second = prev;
        while (second) {
            ListNode *t1 = first->next, *t2 = second->next;
            first->next = second; second->next = t1;
            first = t1; second = t2;
        }
    }
};
class Solution:
    def reorderList(self, head):
        slow, fast = head, head
        while fast.next and fast.next.next: slow = slow.next; fast = fast.next.next
        prev, curr = None, slow.next; slow.next = None
        while curr: nxt = curr.next; curr.next = prev; prev = curr; curr = nxt
        first, second = head, prev
        while second:
            t1, t2 = first.next, second.next
            first.next = second; second.next = t1
            first, second = t1, t2
