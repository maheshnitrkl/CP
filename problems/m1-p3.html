Integer to Roman

Medium 🔗 LeetCode

🔢 Mathematical Foundation

Greedy Subtraction

Roman numerals are fundamentally an additive and subtractive system based on predefined place values (M=1000, CM=900, D=500...).

By always subtracting the largest possible value (Greedy Approach), we guarantee the minimal number of symbols are used.

💡 Approach Progression: Naive → Optimal

Approach 1
Verbose / Hardcoded Conditions

Write massive nested `if/else` or `switch` blocks handling thousands, hundreds, tens, and ones separately.

Time: O(1) Space: O(1)
Pros
  • Straightforward logic
Cons
  • Extremely long and unmaintainable code
  • Prone to typos
Why do we need a better approach?
We can generalize the logic by storing the mapping of values to symbols and using a loop.
Approach 2
Greedy Subtraction

Predefine two arrays/lists mapping values `[1000, 900, 500...]` to symbols `['M', 'CM', 'D'...]`. Repeatedly subtract the largest possible value and append its symbol to the result string.

for each (val, sym) in roman_map:
  while n >= val:
    result += sym
    n -= val
return result
Time: O(1) Space: O(1)
Pros
  • Clean, elegant, and maintainable
Cons
  • None

💻 Implementation

class Solution {
public:
    string intToRoman(int num) {
        vector<pair<int,string>> table = {
            {1000,"M"}, {900,"CM"}, {500,"D"}, {400,"CD"},
            {100,"C"}, {90,"XC"}, {50,"L"}, {40,"XL"},
            {10,"X"}, {9,"IX"}, {5,"V"}, {4,"IV"}, {1,"I"}
        };
        string result;
        for (auto& [val, sym] : table) {
            while (num >= val) {
                result += sym;
                num -= val;
            }
        }
        return result;
    }
};
class Solution:
    def intToRoman(self, num: int) -> str:
        table = [
            (1000, "M"), (900, "CM"), (500, "D"), (400, "CD"),
            (100, "C"), (90, "XC"), (50, "L"), (40, "XL"),
            (10, "X"), (9, "IX"), (5, "V"), (4, "IV"), (1, "I")
        ]
        result = []
        for val, sym in table:
            while num >= val:
                result.append(sym)
                num -= val
        return "".join(result)
